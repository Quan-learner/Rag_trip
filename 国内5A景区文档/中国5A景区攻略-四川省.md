### 1. 九寨沟风景名胜区

**📍 位置：** 阿坝藏族羌族自治州九寨沟县
**🏷 特色：** "九寨归来不看水"，世界自然遗产，以翠海、叠瀑、彩林、雪峰、藏情"五绝"闻名

#### 🗺 推荐旅游路线

**一日游经典路线（景区呈 Y 字形）：**
- 上午：景区入口 → **树正沟**（树正群海、火花海、犀牛海）→ 诺日朗中心（换乘）
- 中午：诺日朗中心午餐
- 下午选择其一：
  - **则查洼沟线**：长海 → 五彩池（九寨沟最小最艳的海子）
  - **日则沟线**：五花海（九寨沟精华）→ 珍珠滩瀑布 → 镜海
- 傍晚：返回诺日朗 → 出景区

**深度两日游：**
- Day 1：树正沟 + 日则沟（重点五花海、珍珠滩）
- Day 2：则查洼沟 + 二次进沟补拍

#### 🍜 附近饮食

| 美食 | 说明 | 推荐 |
|------|------|------|
| 高原牦牛肉火锅 | 散养牦牛肉，无膻味，搭配野生菌汤底 | 沟口藏餐馆 |
| 青稞饼配酥油茶 | 藏式经典搭配，可缓解高原反应 | 藏家乐体验 |
| 九寨野生菌煲 | 天然菌类熬煮，汤汁鲜香 | 景区周边餐厅 |
| 洋芋糍粑 | 当地藏族特色小吃 | 沟口小摊 |

> 💡 **Tips：** 景区内餐饮较贵，建议自带干粮。沟口有大量餐厅和酒店，选择丰富。成都可飞机直达九寨黄龙机场（约40分钟），或自驾经都汶高速（约8小时）。

### 2. 黄龙风景名胜区

**📍 位置：** 阿坝藏族羌族自治州松潘县
**🏷 特色：** 世界自然遗产，以彩池、雪山、峡谷、森林"四绝"著称，有"人间瑶池"之美誉

#### 🗺 推荐旅游路线

**经典一日游：**
- 景区入口 → 索道上山 → 望龙坪观景台 → **五彩池**（黄龙核心精华）→ 争艳彩池 → 盆景池 → 飞瀑流辉 → 迎宾池 → 出口

**徒步路线（约4-5小时）：**
- 全程步行上山 → 五彩池 → 原路或索道下山

#### 🍜 附近饮食

| 美食 | 说明 | 推荐 |
|------|------|------|
| 松潘牦牛肉 | 松潘当地特产，肉质鲜美 | 松潘古城内餐厅 |
| 藏式糌粑 | 青稞面制成，藏族主食 | 川主寺镇 |
| 酥油茶 | 咸香醇厚，高原必备 | 藏家乐 |
| 高原野菜 | 蕨菜、核桃花等时令山珍 | 景区周边 |

> 💡 **Tips：** 海拔3000-3600米，注意高反。通常与九寨沟联游，九黄机场距景区约50公里。建议先游九寨再游黄龙，适应海拔。

### 3. 峨眉山风景名胜区

**📍 位置：** 乐山市峨眉山市
**🏷 特色：** 世界文化与自然双重遗产，中国四大佛教名山之一，金顶日出、云海、佛光、圣灯"四大奇观"

#### 🗺 推荐旅游路线

**一日游速览路线：**
- 报国寺 → 景区观光车至雷洞坪 → 索道上**金顶**（十方普贤金像、华藏寺）→ 原路返回

**两日深度游：**
- Day 1：报国寺 → 伏虎寺 → 清音阁（双桥清音）→ 一线天 → 自然生态猴区 → 洪椿坪（宿山中寺庙/酒店）
- Day 2：早起上金顶看日出 → 华藏寺 → 万年寺（无梁砖殿）→ 下山

**三日全景游：**
- 增加洗象池、雷洞坪夜观圣灯等体验

#### 🍜 附近饮食

| 美食 | 说明 | 推荐 |
|------|------|------|
| 峨眉山豆腐脑 | 当地特色早餐，麻辣鲜香 | 山脚好吃街 |
| 雪魔芋烧鸭 | 峨眉特色菜，魔芋入味 | 报国寺周边 |
| 峨眉鳝丝 | 鳝鱼去骨切丝，口感嫩滑 | 峨眉山市区 |
| 叶儿粑 | 糯米包肉馅，清香软糯 | 市区小吃街 |
| 素斋 | 万年寺、报国寺等寺庙素斋 | 寺庙内 |

> 💡 **Tips：** 金顶海拔3077米，早晚温差大，需带厚外套。猴子可能抢食物，注意保管好随身物品。

### 4. 乐山大佛景区

**📍 位置：** 乐山市市中区
**🏷 特色：** 世界最大的石刻弥勒佛坐像，通高71米，唐代摩崖造像艺术巅峰，"山是一尊佛，佛是一座山"

#### 🗺 推荐旅游路线

**经典路线（约3-4小时）：**
- 景区北门入 → 凌云寺 → **九曲栈道**（近距离仰望大佛）→ 佛脚平台 → 海师洞 → 碑林 → 东门出

**全景路线（约5-6小时）：**
- 陆路游览大佛 + 凌云山 → 乌尤寺 → 麻浩崖墓

**水路观佛：**
- 嘉州渡码头乘游船 → 江上远眺大佛全景（推荐拍照位）

#### 🍜 附近饮食

| 美食 | 说明 | 推荐 |
|------|------|------|
| 乐山甜皮鸭 | 皮酥肉嫩，微甜焦香 | 赵鸭子、纪六娘 |
| 跷脚牛肉 | 乐山名菜，牛肉汤鲜味浓 | 冯四嬢跷脚牛肉 |
| 钵钵鸡 | 冷串串，红油或藤椒味 | 叶婆婆钵钵鸡 |
| 豆腐脑 | 乐山特色，加酥肉粉条 | 嘉州美食街 |
| 乐山烧烤 | 夜宵首选 | 张公桥美食街 |

> 💡 **Tips：** 乐山距成都高铁约1小时。建议与峨眉山联游（相距约30公里）。张公桥美食街是本地人最爱的美食聚集地。

### 5. 青城山-都江堰旅游景区

**📍 位置：** 成都市都江堰市
**🏷 特色：** 青城山为道教四大名山之一，"青城天下幽"；都江堰为世界文化遗产，两千多年前的水利奇迹

#### 🗺 推荐旅游路线

**青城前山（约4-5小时）：**
- 山门 → 月城湖（渡船）→ 索道上山 → **上清宫** → 老君阁（山顶）→ 朝阳洞 → 天师洞 → 五洞天 → 山门

**青城后山（约6-7小时，徒步爱好者）：**
- 泰安古镇 → 五龙沟 → 白云古寨 → 翠映湖 → 返回

**都江堰（约3小时）：**
- 南门进 → 堰功道 → 伏龙观 → **宝瓶口** → 飞沙堰 → **鱼嘴** → 安澜索桥 → 秦堰楼 → 二王庙 → 出

#### 🍜 附近饮食

| 美食 | 说明 | 推荐 |
|------|------|------|
| 白果炖鸡 | 青城山四绝之一 | 山上道观/农家乐 |
| 青城泡菜 | 青城山四绝之一 | 山脚餐馆 |
| 洞天贡茶 | 青城山四绝之一 | 上清宫 |
| 都江堰凉粉 | 酸辣爽滑 | 古城区小吃街 |
| 尤兔头 | 都江堰特色，麻辣鲜香 | 都江堰市区 |

> 💡 **Tips：** 成都出发高铁30分钟直达都江堰站。青城山和都江堰可安排同一天或两天。建议先游都江堰再上青城山。

### 6. 稻城亚丁旅游景区

**📍 位置：** 甘孜藏族自治州稻城县香格里拉镇
**🏷 特色：** "蓝色星球上的最后一片净土"，三神山（仙乃日、央迈勇、夏诺多吉）+ 高原湖泊

#### 🗺 推荐旅游路线

**经典三日游：**
- Day 1：稻城县城 → 红草地（季节性）→ 海子山 → 香格里拉镇（适应海拔）
- Day 2：亚丁景区 → **洛绒牛场** → **牛奶海** → **五色海**（长线徒步，约8-10小时）
- Day 3：亚丁景区 → 冲古寺 → **珍珠海**（短线，看仙乃日倒影）→ 返程

**短线一日游（体力有限者）：**
- 景区入口 → 扎灌崩 → 冲古寺 → 珍珠海 → 返回

#### 🍜 附近饮食

| 美食 | 说明 | 推荐 |
|------|------|------|
| 牦牛肉火锅 | 高原散养牦牛，肉质紧实 | 香格里拉镇 |
| 青稞饼 | 藏族主食，酥脆可口 | 当地藏餐馆 |
| 酥油茶 | 缓解高反，咸香暖胃 | 藏家乐 |
| 松茸炖鸡 | 季节性山珍，鲜美滋补 | 稻城县城 |
| 藏式酸奶 | 浓稠醇厚，配蜂蜜 | 当地小摊 |

> 💡 **Tips：** 景区海拔3700-4700米，高反风险较大，建议提前服用红景天。最佳旅游季节为5-6月（花季）和9-10月（秋色）。可从成都飞稻城亚丁机场（约1小时），再乘车2小时到景区。

### 7. 四姑娘山景区

**📍 位置：** 阿坝藏族羌族自治州小金县
**🏷 特色：** "东方阿尔卑斯"，四座雪峰并立，2024年新晋5A，双桥沟、长坪沟、海子沟三大沟各具特色

#### 🗺 推荐旅游路线

**双桥沟（轻松路线，约4小时）：**
- 观光车直达沟尾 → 依次游览红杉林、**布达拉峰**、四姑娜措、**盆景滩**、人参果坪 → 车行出沟

**长坪沟（徒步/骑马，约6-8小时）：**
- 喇嘛寺 → 枯树滩 → **木骡子**（看四姑娘山全貌）→ 原路返回或骑马

**海子沟（专业徒步，需向导）：**
- 朝山坪 → 大海子 → 花海子

#### 🍜 附近饮食

| 美食 | 说明 | 推荐 |
|------|------|------|
| 藏式火锅 | 牦牛肉+菌类汤底 | 日隆镇 |
| 烤全羊/烤羊排 | 藏族特色大餐 | 藏家乐 |
| 酥油茶+糌粑 | 传统藏式搭配 | 镇上餐馆 |
| 高原苹果 | 小金特产，脆甜多汁 | 路边摊/市场 |

> 💡 **Tips：** 猫鼻梁观景台是免费看四姑娘山全貌的最佳位置，下午日落金山最美。成都出发经巴朗山隧道约4小时车程。

> 💡 **Tips：** 成都乘高铁至剑门关站约2小时。鸟道体验需一定体力，恐高者慎选。晚上可看《剑门长歌》实景演出。

### 10. 阆中古城旅游区

**一日游路线：**
- 古城南门 → **华光楼**（登楼俯瞰古城）→ 中天楼 → **汉桓侯祠（张飞庙）** → 贡院（古代科举考场）→ 川北道署 → 古城街巷漫步 → 嘉陵江边看日落

**两日深度游：**
- Day 1：古城核心景点 + 张飞庙 + 贡院
- Day 2：锦屏山（俯瞰古城全景）→ 南津关古镇 → 嘉陵江游船
