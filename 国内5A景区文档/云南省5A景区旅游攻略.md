## 1. 石林风景区

| 项目 | 信息 |
|------|------|
| **所在城市** | 昆明市石林彝族自治县 |
| **评定年份** | 2007年 |
| **景区类型** | 世界自然遗产（喀斯特地貌） |
| **建议游玩时间** | 3-4小时 |
| **门票** | 130元 |

### 🗺️ 推荐路线

- 大石林（核心景区，石林胜境、剑峰池、望峰亭）→ 小石林（阿诗玛石像）→ 万年灵芝 → 李子园箐

### 🍜 附近美食

| 美食 | 推荐地点 | 说明 |
|------|----------|------|
| 石林卤腐 | 石林县城 | 当地特产 |
| 彝族撒尼烤肉 | 景区周边 | 彝族特色 |
| 野生菌火锅 | 石林/昆明 | 夏秋季最佳 |

### 💡 实用建议

- "阿诗玛的故乡"，彝族撒尼人聚居区
- 从昆明出发约1.5小时车程，可当天往返
- 建议请导游讲解，石头也有故事

## 2. 玉龙雪山景区

| 项目 | 信息 |
|------|------|
| **所在城市** | 丽江市玉龙纳西族自治县 |
| **评定年份** | 2007年 |
| **景区类型** | 高原雪山 |
| **建议游玩时间** | 1天 |
| **门票** | 100元（进山费）+ 索道费另计 |

### 🗺️ 推荐路线

- 甘海子（远眺雪山）→ 大索道（冰川公园，海拔4506m）→ 蓝月谷（强烈推荐！）→ 《印象丽江》演出（可选）

### 🍜 附近美食

| 美食 | 推荐地点 | 说明 |
|------|----------|------|
| 纳西烤鱼 | 丽江古城 | 纳西族特色 |
| 腊排骨火锅 | 丽江象山市场 | 丽江灵魂美食 |
| 鸡豆凉粉 | 丽江古城 | 纳西族传统小吃 |

### 🏨 住宿推荐

- **丽江古城内**：纳西风格客栈（推荐）
- **束河古镇**：比大研古城安静

### 💡 实用建议

- 大索道终点海拔4506m，高反明显，可提前租氧气瓶
- 蓝月谷是玉龙雪山下的蓝绿色湖泊，超出片
- 提前买大索道票，旺季经常售罄

## 6. 普达措国家公园

| 项目 | 信息 |
|------|------|
| **所在城市** | 迪庆藏族自治州香格里拉市 |
| **评定年份** | 2012年 |
| **景区类型** | 高原湿地/原始森林 |
| **建议游玩时间** | 3-5小时 |
| **门票** | 100元（含观光车） |

### 🗺️ 推荐路线

- 入口 → 属都湖（高原湖泊，栈道环湖徒步约1小时）→ 弥里塘亚高山牧场 → 碧塔海（更大的高原湖泊）

### 🍜 附近美食

| 美食 | 推荐地点 | 说明 |
|------|----------|------|
| 牦牛火锅 | 独克宗古城 | 香格里拉必吃 |
| 酥油茶/糌粑 | 藏族餐厅 | 藏族日常饮食 |
| 青稞酒 | 独克宗古城 | 藏族传统酒 |

### 💡 实用建议

- 海拔3500-4100m，注意高反
- 最佳季节：5-6月（花季）、9-10月（秋色）
- 景区内全程栈道+观光车，轻松游览
- 可与独克宗古城、松赞林寺联游

## 8. 火山热海旅游区

| 项目 | 信息 |
|------|------|
| **所在城市** | 保山市腾冲市 |
| **评定年份** | 2016年 |
| **景区类型** | 火山地热 |
| **建议游玩时间** | 半天 |
| **门票** | 火山：40元；热海：78元 |

### 🗺️ 推荐路线

- 上午：热海景区（大滚锅、蛤蟆嘴、珍珠泉、鼓鸣泉）→ 大滚锅煮鸡蛋
- 下午：火山地质公园（大小空山）→ 柱状节理 → 黑鱼河

### 🍜 附近美食

| 美食 | 推荐地点 | 说明 |
|------|----------|------|
| 大救驾 | 腾冲市区 | 炒饵块，腾冲名小吃 |
| 土锅子 | 腾冲市区 | 腾冲版火锅 |
| 稀豆粉 | 腾冲早点 | 当地早餐标配 |
| 大滚锅饵丝 | 热海景区 | 温泉蒸汽煮的饵丝 |

### 💡 实用建议

- 热海大滚锅温度达97℃，可以买鸡蛋在泉眼煮
- 腾冲温泉资源丰富，推荐泡温泉
- 银杏村（11月）和北海湿地也值得去

## 🗺️ 云南5A景区联游推荐路线

### 经典滇西北线（7-8天）
```
昆明（石林）→ 大理（崇圣寺三塔、洱海）→ 丽江（古城、玉龙雪山）→ 香格里拉（普达措）
```

### 滇西腾冲线（4-5天）
```
昆明 → 腾冲（火山热海、和顺古镇、银杏村）
```

### 滇南版纳线（4-5天）
```
昆明 → 西双版纳（热带植物园、野象谷、告庄夜市）
```
