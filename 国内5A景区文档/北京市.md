### 1. 故宫博物院

**📍 位置：** 北京市东城区景山前街4号
**🏷 特色：** 世界现存最大最完整的古代宫殿建筑群，明清两代24位皇帝的皇家宫殿

#### 🗺 推荐旅游路线

**一日游路线：**
- 午门（南门）入院 → 太和门广场（金水桥拍照） → 太和殿（核心大殿，观"金銮宝殿"） → 中和殿 → 保和殿（看云龙大石雕） → 乾清门 → 乾清宫（"正大光明"匾） → 交泰殿 → 坤宁宫 → 御花园（万春亭、千秋亭） → 神武门出 → 景山公园万春亭俯瞰故宫全景（约15分钟登顶），全程约4-5小时

**深度两日游：**
- Day 1（中路+东路）：午门 → 太和殿 → 中和殿 → 保和殿 → 乾清宫 → 坤宁宫 → 御花园 → 神武门出；下午重新从东华门入 → 延禧宫（网红"水晶宫"遗址） → 九龙壁 → 皇极殿 → 珍宝馆（宁寿宫区，含珍妃井） → 钟表馆（奉先殿区，整点有钟表演示） → 东华门出
- Day 2（西路+特展）：午门入 → 武英殿（书画馆特展） → 慈宁宫（雕塑馆） → 寿康宫 → 西六宫（储秀宫、翊坤宫，慈禧寝宫原状陈列） → 军机处 → 冰窖餐厅午餐 → 午门城墙（需预约，俯瞰太和殿广场） → 角楼咖啡/故宫文创店 → 神武门出

#### 🍜 附近饮食

| 美食 | 说明 | 推荐 |
|------|------|------|
| 故宫冰窖餐厅 | 宫廷风味简餐，环境古朴，招牌万福肉 | 故宫内冰窖（慈宁宫旁） |
| 四季民福烤鸭 | 北京烤鸭经典，皮脆肉嫩，可观故宫东华门 | 故宫店（南池子大街11号） |
| 角楼咖啡 | 故宫官方文创咖啡，"康熙最爱巧克力"等特色饮品 | 神武门外东侧 |

> 💡 **Tips：** ①门票必须提前在"故宫博物院"官网/小程序实名预约，旺季（4-10月）建议提前7天抢票；②每周一闭馆（法定节假日除外）；③建议从南侧午门进、北侧神武门出，单向游览；④地铁1号线天安门东站B口出，步行约10分钟至午门；⑤馆内禁止使用自拍杆和三脚架。

### 2. 天坛公园

**📍 位置：** 北京市东城区天坛内东里7号
**🏷 特色：** 明清两代皇帝祭天祈谷的圣地，祈年殿是中国古代建筑艺术的巅峰之作

#### 🗺 推荐旅游路线

**一日游路线：**
- 南门入 → 圜丘坛（天心石，感受回音效果） → 皇穹宇（回音壁，轻声说话对面可闻） → 丹陛桥（长360米的海墁大道，连接南北两组建筑） → 祈年殿（三重檐圆形大殿，标志建筑，拍照C位） → 皇乾殿 → 七星石 → 长廊 → 北门出，全程约3小时

**深度两日游：**
- Day 1（中轴线精华）：南门 → 圜丘坛 → 皇穹宇 → 丹陛桥 → 祈年殿 → 北门出；傍晚在天坛公园西侧外环路散步，感受老北京生活气息
- Day 2（外围古柏区+深度文化）：东门入 → 宰牲亭 → 神厨（天坛文物展） → 长廊（老北京人晨练、唱戏的场所） → 斋宫（皇帝祭天前斋戒之处） → 双环万寿亭 → 西天门 → 月季园（春季月季花开） → 南门出

#### 🍜 附近饮食

| 美食 | 说明 | 推荐 |
|------|------|------|
| 老磁器口豆汁店 | 地道北京豆汁配焦圈，酸香醇厚，老北京早餐标配 | 天坛东门对面磁器口 |
| 都一处烧麦 | 始于乾隆年间，烧麦皮薄馅大，三鲜馅最受欢迎 | 前门大街36号（距天坛约2公里） |
| 天坛福饮 | 天坛公园内文创饮品店，特色"祈年殿"造型甜品 | 祈年殿西侧 |

> 💡 **Tips：** ①公园联票34元（含祈年殿、回音壁、圜丘），普通门票15元；②建议早上7点前到达，可观赏老北京人晨练、踢毽、合唱的热闹场景；③回音壁实际效果受游客嘈杂影响较大，建议工作日前往；④地铁5号线天坛东门站A口出。

### 3. 颐和园

**📍 位置：** 北京市海淀区新建宫门路19号
**🏷 特色：** 中国现存最大的皇家园林，昆明湖与万寿山构成的"山水画卷"，世界文化遗产

#### 🗺 推荐旅游路线

**一日游路线：**
- 东宫门入 → 仁寿殿（听政大殿） → 德和园（大戏楼，慈禧看戏处） → 玉澜堂（光绪帝被囚处） → 长廊（728米，14000余幅彩画，世界最长画廊） → 排云殿 → 佛香阁（登顶俯瞰昆明湖全景，核心打卡点） → 石舫（清晏舫） → 苏州街（仿江南水镇） → 北宫门出，全程约4-5小时

**深度两日游：**
- Day 1（前山主景区）：东宫门 → 仁寿殿 → 德和园 → 玉澜堂 → 长廊 → 排云殿 → 佛香阁 → 石舫 → 北宫门出
- Day 2（后山后湖+西堤）：北宫门入 → 苏州街 → 谐趣园（"园中之园"，小巧精致） → 后山古建筑群遗址 → 乘船游昆明湖（铜牛码头 → 南湖岛，十七孔桥拍照） → 西堤六桥（仿杭州西湖苏堤，春季桃柳最美） → 南如意门出

#### 🍜 附近饮食

| 美食 | 说明 | 推荐 |
|------|------|------|
| 听鹂馆饭庄 | 宫廷风味餐厅，始于乾隆年间，招牌"万寿无疆"席 | 颐和园内长廊西侧 |
| 西贝莜面村（颐和园店） | 西北风味，莜面鱼鱼、羊肉汤暖身 | 北宫门外步行5分钟 |
| 麦当劳/肯德基 | 快餐选择，方便快捷 | 北宫门外商业街 |

> 💡 **Tips：** ①旺季（4-10月）联票60元（含佛香阁、苏州街等），淡季联票50元；②昆明湖游船另收费（电动船约80元/小时）；③建议从东宫门入、北宫门出，不走回头路；④西堤六桥在3-4月桃花盛开时最美；⑤地铁4号线北宫门站D口出即到。

### 4. 八达岭-慕田峪长城旅游区

**📍 位置：** 北京市延庆区G6京藏高速58号出口 / 怀柔区渤海镇
**🏷 特色：** 八达岭是长城最雄伟的段落，慕田峪则以秀美著称，各有千秋

#### 🗺 推荐旅游路线

**八达岭一日游路线：**
- 关城广场（入口） → 北一楼 → 北二楼 → 北三楼 → 北四楼（好汉坡，拍照打卡） → 北八楼（最高点，海拔888米，视野开阔） → 原路返回或继续至北十一楼 → 出口，全程约3-4小时

**慕田峪一日游路线：**
- 景区大门 → 乘摆渡车至缆车下站 → 缆车上山至14号敌楼 → 14号 → 20号敌楼（好汉坡，慕田峪最高点） → 返回14号敌楼 → 缆车下山，全程约4小时

**深度两日游：**
- Day 1（八达岭）：早起前往八达岭 → 北线登顶（北八楼） → 参观中国长城博物馆 → 住延庆/昌平农家院
- Day 2（慕田峪）：上午前往慕田峪 → 缆车上14号敌楼 → 西行至20号敌楼 → 下山后品尝怀柔虹鳟鱼 → 返回市区

#### 🍜 附近饮食

| 美食 | 说明 | 推荐 |
|------|------|------|
| 延庆火勺 | 延庆特色面食，外酥里嫩，夹驴肉或酱肉 | 八达岭商业街小摊 |
| 怀柔虹鳟鱼 | 清蒸或烤制，肉质鲜嫩，怀柔第一招牌 | 慕田峪附近鱼师傅（渤海镇） |
| 长城脚下农家菜 | 柴鸡炖蘑菇、贴饼子、山野菜 | 八达岭/慕田峪周边农家乐 |

> 💡 **Tips：** ①八达岭需提前在"长城内外"微信公众号预约；②八达岭建议乘S2线市郊铁路（黄土店站发车，6元，1小时）避免堵车；③慕田峪比八达岭人少，外国游客多，体验更好；④春秋最佳，冬季风大极冷；⑤八达岭停车场旺季常满，建议公共交通。

### 5. 明十三陵景区

**📍 位置：** 北京市昌平区十三陵镇
**🏷 特色：** 明朝十三位皇帝的陵寝所在地，规模宏大，神道石刻精美绝伦

#### 🗺 推荐旅游路线

**一日游路线：**
- 神路（总神道，石象生、石骆驼、石麒麟等18对石雕，全长7公里，参观约1小时） → 定陵（唯一发掘的皇陵，地下宫殿深27米，可入地宫参观，约1.5小时） → 长陵（朱棣陵寝，祾恩殿是中国最大的楠木大殿，约1小时） → 返程，全程约5小时

**深度两日游：**
- Day 1（神道+定陵+长陵）：神路 → 定陵地宫 → 长陵祾恩殿 → 住昌平城区
- Day 2（周边陵寝+自然风光）：昭陵（修复最完整的陵寝） → 康陵（春季赏杏花） → 银山塔林景区（辽金时期古塔群，登高远眺十三陵水库） → 返程

#### 🍜 附近饮食

| 美食 | 说明 | 推荐 |
|------|------|------|
| 十三陵炸糕 | 外皮金黄酥脆，豆沙馅香甜，老北京传统小吃 | 十三陵景区入口摊位 |
| 昌平草莓宴 | 草莓采摘+草莓主题菜肴（11-5月草莓季） | 昌平草莓采摘园（十三陵镇周边） |
| 虹鳟鱼一条沟 | 烤虹鳟鱼、鲟鱼，山泉水养殖 | 昌平区长陵镇麻峪房村 |

> 💡 **Tips：** ①联票130元（含神路+长陵+定陵+昭陵），单景点门票约30-65元；②建议自驾或包车，景区间距离较远；③定陵地宫阴冷，夏季也建议带外套；④德胜门乘345路/872路公交至昌平，转314路可达。

### 6. 恭王府景区

**📍 位置：** 北京市西城区什刹海前海西街17号
**🏷 特色：** 清代规模最大的王府，和珅旧宅，"一座恭王府，半部清代史"

#### 🗺 推荐旅游路线

**一日游路线：**
- 大门 → 银安殿（王府正殿，银銮殿） → 嘉乐堂 → 葆光室 → 锤楼 → 后罩楼（藏宝楼，和珅藏宝处，窗棂花纹各异对应不同宝物） → 西洋门（中西合璧的汉白玉石门） → 独乐峰 → 沁秋亭（流杯亭） → 福字碑（康熙御笔"天下第一福"，排队摸福） → 蝠池 → 大戏楼（全封闭式室内戏楼，声学设计精妙） → 东门出，全程约2-3小时

**深度半日游+什刹海联动：**
- 上午：恭王府完整游览（路线同上）
- 下午：什刹海胡同游（坐人力三轮车穿梭胡同） → 银锭桥（"银锭观山"燕京小八景） → 烟袋斜街 → 后海酒吧街（傍晚至夜间）

#### 🍜 附近饮食

| 美食 | 说明 | 推荐 |
|------|------|------|
| 烤肉季 | 清真烤羊肉，始于1848年，百年老字号 | 什刹海前海东沿14号 |
| 姚记炒肝 | 地道北京炒肝配包子，"要想吃炒肝，鼓楼一拐弯" | 鼓楼东大街311号 |
| 护国寺小吃 | 豆汁、面茶、驴打滚、艾窝窝等京味小吃合集 | 护国寺街（距恭王府步行10分钟） |

> 💡 **Tips：** ①门票40元，需提前预约；②"福字碑"是游客最集中的景点，建议开门即入，先奔福字碑；③后罩楼的窗棂花纹据说暗含和珅藏宝密码，值得细看；④地铁6号线北海北站B口出，步行约8分钟；⑤可与什刹海、南锣鼓巷串联游览。

### 7. 北京奥林匹克公园

**📍 位置：** 北京市朝阳区北辰路
**🏷 特色：** 2008年北京奥运会主场馆所在地，鸟巢与水立方已成为北京新地标

#### 🗺 推荐旅游路线

**一日游路线：**
- 南门入 → 奥林匹克景观大道（中轴线步行道） → 鸟巢（国家体育场，外部拍照为主，内部参观约40分钟） → 水立方（国家游泳中心，可参观嬉水乐园） → 玲珑塔（奥运转播塔） → 奥林匹克塔（登塔俯瞰中轴线，高246米） → 北顶娘娘庙（古庙与现代建筑的奇妙共存） → 森林公园南门 → 奥海（人工湖）散步 → 北门出，全程约4-5小时

**深度半日+夜间游：**
- 下午：鸟巢+水立方参观 → 森林公园骑行（园内可租自行车，沿跑道骑行约10公里）
- 夜晚：返回鸟巢/水立方欣赏夜景灯光（水立方常亮蓝光，鸟巢内部红光透出，极为壮观） → 奥林匹克塔夜景

#### 🍜 附近饮食

| 美食 | 说明 | 推荐 |
|------|------|------|
| 新奥购物中心美食 | 各类中外餐饮，海底捞、眉州东坡等 | 鸟巢东侧新奥购物中心 |
| 眉州东坡酒楼 | 川菜连锁，东坡肘子、宫保鸡丁 | 新奥购物中心B1层 |
| 兰熊鲜奶 | 网红酸奶饮品，水果捞和鲜奶昔 | 奥林匹克公园周边 |

> 💡 **Tips：** ①公园免费，鸟巢50元、水立方30元、奥林匹克塔128元；②水立方已改造为"冰立方"（冬奥遗产），可参观冰壶场地；③夜间灯光效果最佳，建议傍晚前往；④地铁8号线奥林匹克公园站或15号线奥林匹克公园站。

### 8. 圆明园遗址公园

**📍 位置：** 北京市海淀区清华西路28号
**🏷 特色：** "万园之园"的遗址，西洋楼残柱是中国人集体记忆的符号

#### 🗺 推荐旅游路线

**一日游路线：**
- 绮春园宫门入 → 鉴碧亭 → 正觉寺（圆明园展览馆，了解全盛时期模型） → 绮春园遗址 → 福海（园内最大水域，可乘船） → 蓬岛瑶台遗址 → 西洋楼景区（大水法遗址、远瀛观残柱、海晏堂、方外观、谐奇趣，核心打卡区，约1小时） → 黄花阵迷宫（复原的欧式迷宫） → 东门出，全程约3-4小时

**深度两日游：**
- Day 1（西部遗址+西洋楼）：绮春园宫门 → 正觉寺展览 → 西洋楼遗址区（重点参观） → 福海泛舟 → 东门出

### 9. 北京(通州)大运河文化旅游景区

**📍 位置：** 北京市通州区大成街1号（大光楼码头区域）
**🏷 特色：** 京杭大运河北京段核心区域，千年运河文化与现代城市副中心的融合

#### 🗺 推荐旅游路线

**一日游路线：**
- 大光楼码头（登船起点） → 乘船游览运河（约40分钟，两岸城市副中心风光） → 月亮河观景平台 → 大运河森林公园（运河两岸生态廊道，骑行/步行） → 运河文化广场 → 燃灯塔及文庙（通州古城标志，始建于北周） → 西海子公园（李卓吾墓） → 三庙一塔景区（儒释道三教合一，全国罕见） → 千荷泻露桥拍照 → 返程，全程约4-5小时

**深度两日游：**
- Day 1（运河主景区）：大光楼码头乘船 → 运河文化广场 → 大运河森林公园骑行（全程约12公里绿道） → 月岛闻莺观景 → 运河夜景灯光秀（周末/节假日）
- Day 2（通州古城+副中心）：燃灯塔 → 三庙一塔景区 → 西海子公园 → 北京城市副中心规划展览馆（了解新城规划） → 大运河博物馆（运河文化全面展示） → 返程
