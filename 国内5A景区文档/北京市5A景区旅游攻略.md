## 1. 故宫博物院

| 项目 | 信息 |
|------|------|
| **所在区域** | 东城区 |
| **评定年份** | 2007年 |
| **景区类型** | 世界文化遗产、皇家宫殿 |
| **建议游玩时间** | 3-5小时 |
| **门票** | 旺季（4-10月）：60元；淡季：40元 |

### 🗺️ 推荐路线

**经典中轴线（2-3小时）：**
- 午门 → 太和殿 → 中和殿 → 保和殿 → 乾清宫 → 交泰殿 → 坤宁宫 → 御花园 → 神武门

**深度游（4-5小时）：**
- 午门 → 武英殿（书画馆）→ 太和殿 → 中和殿 → 保和殿 → 奉先殿（钟表馆）→ 东六宫 → 延禧宫 → 珍宝馆 → 九龙壁 → 乾清宫 → 御花园 → 神武门

### 🍜 附近美食

| 美食 | 推荐地点 | 说明 |
|------|----------|------|
| 四季民福烤鸭 | 故宫旁南池子店 | 边吃烤鸭边看故宫角楼 |
| 故宫冰窖餐厅 | 故宫内冰窖 | 皇家建筑里用餐 |
| 护国寺小吃 | 护国寺街 | 豆汁、焦圈、驴打滚等老北京小吃 |

### 💡 实用建议

- 必须提前在"故宫博物院"微信小程序预约，旺季一票难求
- 周一闭馆（法定节假日除外）
- 从南门（午门）进，北门（神武门）出，不走回头路

## 3. 颐和园

| 项目 | 信息 |
|------|------|
| **所在区域** | 海淀区 |
| **评定年份** | 2007年 |
| **景区类型** | 世界文化遗产、皇家园林 |
| **建议游玩时间** | 3-4小时 |
| **门票** | 旺季：30元；淡季：20元 |

### 🗺️ 推荐路线

- 东宫门 → 仁寿殿 → 德和园 → 长廊（728米，世界最长画廊）→ 排云殿 → 佛香阁（登顶俯瞰昆明湖）→ 石舫 → 苏州街 → 北宫门出

### 🍜 附近美食

| 美食 | 推荐地点 | 说明 |
|------|----------|------|
| 听鹂馆饭庄 | 颐和园内 | 皇家御膳，面朝昆明湖 |
| 西苑美食街 | 颐和园北门附近 | 各类餐厅集中 |

### 💡 实用建议

- 佛香阁是制高点，建议登顶
- 昆明湖可坐船（夏季）
- 日落时分的十七孔桥"金光穿洞"非常壮观（冬至前后最佳）

## 4. 八达岭—慕田峪长城旅游区

| 项目 | 信息 |
|------|------|
| **所在区域** | 延庆区/怀柔区 |
| **评定年份** | 2007年 |
| **景区类型** | 世界文化遗产 |
| **建议游玩时间** | 半天-1天 |
| **门票** | 八达岭：40元；慕田峪：45元 |

### 🗺️ 推荐路线

**八达岭（人多但设施完善）：**
- 北门锁钥 → 北一楼 → 北八楼（好汉坡）→ 原路返回或南线

**慕田峪（人少景美，推荐）：**
- 摆渡车到南检票口 → 缆车上山 → 14号敌楼 → 徒步到20号敌楼 → 原路返回

### 🍜 附近美食

| 美食 | 推荐地点 | 说明 |
|------|----------|------|
| 虹鳟鱼 | 慕田峪附近农家院 | 怀柔特色，烤或炖 |
| 栗子宴 | 怀柔农家院 | 当地板栗入菜 |

### 💡 实用建议

- **强烈推荐慕田峪**：人少、景美、体验好，外国游客多
- 八达岭交通方便（S2线火车/877公交），但人山人海
- 春秋季最佳，夏天暴晒，冬天结冰路滑

| 美食 | 推荐地点 | 说明 |
|------|----------|------|
| 春饼宴 | 十三陵附近康陵村 | 当地特色农家菜 |

- 可与八达岭长城安排在同一天（相距约30分钟车程）

| 美食 | 推荐地点 | 说明 |
|------|----------|------|
| 护国寺小吃 | 护国寺街 | 豆汁、焦圈、面茶等 |

- 可与什刹海、南锣鼓巷联游

| 美食 | 推荐地点 | 说明 |
|------|----------|------|

| **景区类型** | 历史遗址公园 |
| **建议游玩时间** | 2-3小时 |
| **门票** | 10元（联票25元） |

- 绮春园宫门 → 鉴碧亭 → 凤麟洲 → 西洋楼遗址（大水法、海晏堂、远瀛观）→ 福海 → 全景沙盘 → 东门出

- 荷花季（7-8月）非常美
- 可与颐和园安排在同一天（相距约3公里）

## 🗺️ 北京5A景区联游推荐路线

### 经典三日游
```
Day 1: 故宫 → 天坛 → 前门大街（烤鸭）
Day 2: 慕田峪长城 → 明十三陵 → 奥林匹克公园（夜景）
Day 3: 颐和园 → 圆明园 → 恭王府 → 什刹海
```

### 经典五日深度游
```
Day 1: 天安门 → 故宫 → 景山公园 → 什刹海
Day 2: 慕田峪长城 → 十三陵
Day 3: 颐和园 → 圆明园 → 北大/清华
Day 4: 天坛 → 恭王府 → 南锣鼓巷 → 国子监
Day 5: 通州大运河 → 鸟巢/水立方 → 798艺术区
```
