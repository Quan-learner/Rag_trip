## 1. 长隆旅游度假区

**📍位置**：广东省广州市番禺区大石街道，距广州市中心约15公里。

**🏷特色**：亚洲最大的综合性主题旅游度假区，集野生动物世界、欢乐世界、水上乐园、大马戏于一体，是华南地区亲子游首选目的地。

**🗺推荐旅游路线**：

- **一日游（欢乐世界）**：垂直过山车 → 十环过山车 → 超级大摆锤 → 飞马家庭过山车 → 摩托过山车 → 水上乐园（夏季）→ 大马戏（夜间）
- **一日游（野生动物世界）**：小火车穿越五大洲动物区 → 长颈鹿投喂 → 大熊猫中心 → 白虎山 → 青龙山 → 考拉园 → 大象传奇表演
- **深度游（3日）**：D1 长隆野生动物世界；D2 长隆欢乐世界 + 水上乐园（夏季）；D3 长隆大马戏 + 飞鸟乐园

**🍜附近饮食**：

| 美食 | 推荐店铺 | 人均 | 说明 |
|-----|---------|-----|------|
| 粤式早茶 | 长隆酒店麒麟中餐厅 | 80元 | 虾饺、烧麦、肠粉，正宗粤式早茶 |
| 番禺猪杂粥 | 番禺标记猪杂粥（大石店） | 30元 | 猪杂新鲜，粥底绵滑 |
| 顺德双皮奶 | 民信老铺（番禺分店） | 18元 | 奶香浓郁，入口即化 |
| 潮汕牛肉火锅 | 八合里海记（番禺万达店） | 90元 | 现切鲜牛肉涮火锅 |

**💡Tips**：
- 长隆大马戏强烈推荐，全球顶级马戏表演，需单独购票（约300元）。
- 野生动物世界小火车和空中缆车免费乘坐，建议都体验。
- 7-8月水上乐园开放，可与欢乐世界组合游玩。
- 建议住长隆酒店，有专属通道入园且可看白虎。

## 2. 华侨城

**📍位置**：广东省深圳市南山区深南大道，深圳市中心区域。

**🏷特色**：集世界之窗（全球微缩景观）、锦绣中华（中国微缩景观）、欢乐谷（主题乐园）于一体的大型文化旅游区，是深圳地标性景区。

**🗺推荐旅游路线**：

- **一日游（欢乐谷）**：雪山飞龙 → 完美风暴 → 激流勇进 → 太空梭 → 欢乐时光主题区 → 魔幻城堡 → 玛雅水公园（夏季）
- **一日游（世界之窗）**：亚洲区 → 欧洲区 → 非洲区 → 美洲区 → 大洋洲区 → 阿尔卑斯冰雪世界 → 埃菲尔铁塔（夜间灯光秀）
- **深度游（2日）**：D1 欢乐谷全天；D2 世界之窗 + 锦绣中华 → 深圳湾公园日落

**🍜附近饮食**：

| 美食 | 推荐店铺 | 人均 | 说明 |
|-----|---------|-----|------|
| 潮汕牛肉丸 | 八合里海记（南山店） | 90元 | 现打牛肉丸，弹牙爽口 |
| 广式肠粉 | 银记肠粉（华侨城店） | 18元 | 布拉肠粉，滑嫩鲜美 |
| 深圳椰子鸡 | 椰客（南山店） | 80元 | 椰子水煮文昌鸡，清甜鲜美 |
| 粤式糖水 | 满记甜品（华侨城店） | 30元 | 杨枝甘露、芒果班戟 |

**💡Tips**：
- 欢乐谷夜场票更便宜且氛围更好，推荐下午入园玩到闭园。
- 世界之窗夜间灯光秀和烟花表演（周末/节假日）非常精彩。
- 华侨城创意文化园（OCT-LOFT）在旁边，文艺青年打卡地。
- 深圳湾公园可步行看对岸香港，日落时分最美。

## 3. 白云山

**📍位置**：广东省广州市白云区广园中路，广州市中心北部。

**🏷特色**：广州市的"城市绿肺"，自古就有"羊城第一秀"之称，山不高（主峰摩星岭382米）但景色秀丽，是广州市民休闲登山首选。

**🗺推荐旅游路线**：

- **一日游**：南门（云台花园）→ 蒲谷 → 能仁寺 → 摩星岭（登顶远眺广州全景）→ 鸟鸣春 → 明珠楼 → 桃花涧 → 飞鹅岭 → 西门出
- **深度游（2日）**：D1 白云山全景（同上）；D2 越秀公园（五羊雕塑）→ 陈家祠 → 北京路步行街 → 珠江夜游

**🍜附近饮食**：

| 美食 | 推荐店铺 | 人均 | 说明 |
|-----|---------|-----|------|
| 广州早茶 | 白云山脚广州酒家 | 80元 | 虾饺、叉烧包、蛋挞，经典粤式早茶 |
| 白云山泉水豆腐花 | 山上豆腐花摊 | 8元 | 山泉水制豆腐花，滑嫩清甜 |
| 沙河粉 | 沙河粉村（白云山附近） | 20元 | 广州传统河粉，炒或汤均可 |
| 煲仔饭 | 超记煲仔饭（白云区） | 25元 | 锅巴焦脆，腊味飘香 |

**💡Tips**：
- 白云山门票仅5元，性价比极高，适合晨练和休闲。
- 摩星岭是白云山最高点，可360度俯瞰广州城。
- 云台花园是白云山脚的欧式花园，适合拍照。
- 建议早上上山，避开中午高温，带足饮用水。

## 4. 丹霞山

**📍位置**：广东省韶关市仁化县，距韶关市区约45公里。

**🏷特色**：世界自然遗产、世界地质公园，丹霞地貌的命名地，以赤壁丹崖为特色，680多座红色砂砾岩山峰构成壮观的丹霞景观。

**🗺推荐旅游路线**：

- **一日游**：阳元石（自然奇观）→ 通泰桥 → 翔龙湖 → 阴元石 → 长老峰（登顶看日出/日落）→ 锦江竹筏漂流 → 仙居岩
- **深度游（2日）**：D1 丹霞山全景（同上）；D2 锦江画廊竹筏漂流 → 丹霞山地质博物馆 → 南华禅寺（六祖惠能）→ 韶关风采楼

**🍜附近饮食**：

| 美食 | 推荐店铺 | 人均 | 说明 |
|-----|---------|-----|------|
| 爆炒山坑螺 | 仁化农家乐 | 30元 | 山溪石螺爆炒，鲜辣过瘾 |
| 丹霞臭豆腐 | 丹霞山景区小吃街 | 10元 | 当地特色臭豆腐 |
| 韶关铜勺饼 | 韶关老街铜勺饼摊 | 8元 | 米浆铜勺炸制，酥脆金黄 |
| 南华素斋 | 南华禅寺素斋馆 | 25元 | 六祖道场素食 |

**💡Tips**：
- 阳元石和阴元石是丹霞山的自然奇观，造型惟妙惟肖。
- 长老峰看日出需凌晨5点前出发，建议住景区内。
- 锦江竹筏漂流约1小时，沿途丹霞风光尽收眼底。
- 丹霞山与南华禅寺可安排在同一天，相距约20公里。

## 5. 开平碉楼

**📍位置**：广东省江门市开平市，距江门市区约60公里。

**🏷特色**：世界文化遗产，是中国乡土建筑中集防卫、居住和中西建筑艺术于一体的多层塔楼式建筑，现存1833座，见证华侨文化与近代历史。

**🗺推荐旅游路线**：

- **一日游**：自力村碉楼群（最经典的碉楼群落）→ 立园（旅美华侨谢维立的私人园林）→ 马降龙古村落（竹林中的碉楼群）→ 赤坎古镇（骑楼老街）
- **深度游（2日）**：D1 开平碉楼全景（同上）；D2 赤坎古镇深度游 → 台山海侨东南亚风情园 → 上下川岛（滨海度假）

**🍜附近饮食**：

| 美食 | 推荐店铺 | 人均 | 说明 |
|-----|---------|-----|------|
| 开平黄鳝饭 | 赤坎黄鳝饭老店 | 25元 | 砂锅焖饭配黄鳝，香气扑鼻 |
| 台山黄油蟹 | 台山海鲜酒楼 | 100元 | 台山特产膏蟹，蟹膏金黄 |
| 恩平濑粉 | 恩平濑粉店 | 12元 | 米浆制成粉条，配猪骨汤 |
| 赤坎豆腐角 | 赤坎古镇小吃摊 | 8元 | 油炸豆腐角，外酥内嫩 |

**💡Tips**：
- 自力村是碉楼最集中的区域，《让子弹飞》取景地。
- 立园是碉楼与园林的完美结合，中西合璧，值得细看。
- 赤坎古镇正在修缮中，骑楼老街保留了民国风貌。
- 秋季（10-11月）稻田金黄时拍碉楼最美。

## 6. 罗浮山

**📍位置**：广东省惠州市博罗县，距惠州市区约60公里。

**🏷特色**：道教十大洞天之第七洞天，岭南第一山，东晋葛洪在此炼丹著书，苏东坡曾留下"罗浮山下四时春，卢橘杨梅次第新"的名句。

**🗺推荐旅游路线**：

- **一日游**：冲虚古观（葛洪炼丹处）→ 会仙桥 → 朱明洞 → 白莲池 → 飞瀑名泉 → 黄龙洞 → 华首寺 → 登飞云顶（主峰1296米）
- **深度游（2日）**：D1 罗浮山全景（同上）；D2 博罗哈施塔特小镇 → 惠州西湖 → 红花湖骑行

**🍜附近饮食**：

| 美食 | 掗荐店铺 | 人均 | 说明 |
|-----|---------|-----|------|
| 罗浮山豆腐花 | 山脚豆腐花摊 | 8元 | 山泉水制豆腐花，滑嫩清甜 |
| 罗浮山酥醪菜 | 博罗农家乐 | 35元 | 罗浮山特产蔬菜，清甜爽口 |
| 博罗酿豆腐 | 博罗酿豆腐店 | 25元 | 客家酿豆腐，嫩滑多汁 |
| 东江盐焗鸡 | 惠州东江盐焗鸡店 | 45元 | 客家名菜，皮脆肉嫩 |

**💡Tips**：
- 冲虚古观是罗浮山最古老的道观，葛洪在此著《肘后备急方》（青蒿素灵感来源）。
- 飞云顶登顶约需3-4小时，体力不佳者可走到半山返回。
- 罗浮山是岭南中草药宝库，山上有百草油等特产。
- 夏季山上凉爽，是避暑好去处。

## 7. 西樵山

**📍位置**：广东省佛山市南海区西樵镇，距佛山市区约20公里。

**🏷特色**：南粤名山之一，以"双肩石器"闻名的考古遗址，黄飞鸿故里，也是南拳南狮的发源地，山上有世界最高的观音坐像。

**🗺推荐旅游路线**：

- **一日游**：南海观音文化苑（世界最高观音坐像）→ 白云洞 → 天湖公园 → 石燕岩 → 翠岩 → 黄飞鸿狮艺武术馆（武术表演）→ 宝峰寺
- **深度游（2日）**：D1 西樵山全景（同上）；D2 佛山祖庙（黄飞鸿纪念馆）→ 南风古灶 → 岭南天地 → 佛山创意产业园

**🍜附近饮食**：

| 美食 | 推荐店铺 | 人均 | 说明 |
|-----|---------|-----|------|
| 西樵大饼 | 西樵大饼老店 | 10元 | 西樵特产大饼，松软香甜 |
| 佛山盲公丸 | 应记面家（佛山祖庙店） | 20元 | 手打鱼丸，弹牙鲜美 |
| 顺德鱼生 | 顺德鱼生专门店 | 60元 | 顺德特色生鱼片，薄如蝉翼 |
| 均安蒸猪 | 均安蒸猪专门店 | 80元 | 顺德名菜，整猪蒸制 |

**💡Tips**：
- 黄飞鸿狮艺武术馆每天有免费武术和醒狮表演，非常精彩。
- 南海观音坐像高61.9米，是世界最高的观音铜坐像。
- 西樵山与佛山祖庙可串联游览，了解黄飞鸿文化。
- 顺德美食之都就在旁边，美食爱好者可延伸到顺德觅食。

| 东江酿豆腐 | 惠州东江酿豆腐店 | 25元 | 客家三大名菜之一 |

**💡Tips**：

## 9. 孙中山故里

**📍位置**：广东省中山市南朗街道翠亨村，距中山市区约20公里。

**🏷特色**：孙中山先生的故乡，保存有孙中山故居、纪念馆等遗址，展示了中国民主革命先行者的生平事迹和翠亨村的历史风貌。

**🗺推荐旅游路线**：

- **一日游**：孙中山故居（中西合璧的民居）→ 孙中山纪念馆 → 翠亨村（传统岭南民居）→ 中山城（影视基地）→ 孙中山纪念堂 → 中山步行街
- **深度游（2日）**：D1 孙中山故里全景（同上）；D2 中山詹园（岭南园林）→ 中山温泉 → 岐江公园夜景

**🍜附近饮食**：

| 美食 | 推荐店铺 | 人均 | 说明 |
|-----|---------|-----|------|
| 石岐乳鸽 | 石岐乳鸽皇（中山总店） | 40元 | 中山第一名菜，皮脆肉嫩 |

## 16. 地下河

**📍位置**：广东省清远市连州市东陂镇，距清远市区约150公里。
