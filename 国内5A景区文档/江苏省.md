# 江苏省 5A级旅游景区攻略

## 省份简介

江苏省，简称"苏"，位于中国大陆东部沿海中心、长江下游，东濒黄海，东南与浙江和上海毗邻，西接安徽，北接山东。江苏是中国古代文明的发祥地之一，拥有"鱼米之乡"的美誉。省会南京是中国四大古都之一，苏州园林、南京明孝陵等多处世界文化遗产彰显着深厚的历史底蕴。江苏经济发达，GDP长期位居全国前列，是中国综合发展水平最高的省份之一。全省13个地级市全部进入全国百强市，创造了独特的"苏南模式"。江苏旅游资源极为丰富，截至2024年共有26家国家5A级旅游景区，数量位居全国前列，涵盖古典园林、水乡古镇、历史文化名城、自然山水、主题乐园等多种类型。

## 1. 钟山-中山陵

### 📍位置
南京市玄武区石象路7号，位于南京城东紫金山南麓。

### 🏷特色
钟山风景区是南京的"城市绿肺"和精神地标，总面积约31平方公里，融山、水、城、林于一体。中山陵是伟大的民主革命先行者孙中山先生的陵寝，由著名建筑师吕彦直设计，整体呈"自由钟"形，寓意"唤起民众"。从博爱坊到祭堂，共有392级台阶，象征当时中国3.92亿人口。祭堂内安放着孙中山先生坐像，庄严肃穆。钟山风景区内还汇聚了明孝陵（世界文化遗产）、灵谷寺、美龄宫、音乐台等众多名胜。明孝陵是明太祖朱元璋的陵墓，神道石刻群气势恢宏。春赏梅花（梅花山为全国四大梅园之首），秋看银杏（石象路被誉为"南京最美600米"），四季皆景。

### 🗺推荐旅游路线

**一日游路线：**
中山陵（博爱坊→陵门→碑亭→祭堂）→ 音乐台（午餐）→ 明孝陵（神道→明楼→宝顶）→ 美龄宫 → 灵谷寺

**深度游路线：**
Day 1：中山陵 → 音乐台 → 美龄宫 → 明孝陵 → 石象路日落
Day 2：灵谷寺 → 灵谷塔登高 → 中山植物园 → 紫金山天文台
Day 3：总统府 → 南京1912街区 → 长江大桥 → 阅江楼

### 🍜附近饮食

| 美食 | 说明 | 推荐店铺 |
|------|------|---------|
| 南京盐水鸭 | 南京第一名菜，皮白肉嫩，鲜香可口 | 韩复兴、章云板鸭 |
| 鸭血粉丝汤 | 南京经典小吃，鸭血嫩滑，粉丝入味 | 回味鸭血粉丝汤 |
| 桂花糖芋苗 | 南京传统甜品，桂花飘香，芋头软糯 | 南京大排档 |

### 💡Tips
- 中山陵免费但需提前在"钟山风景区"公众号预约，限流管理
- 建议穿舒适运动鞋，台阶较多
- 景区面积大，可乘坐景区观光车（20元/次）
- 明孝陵神道建议从石象路方向进入，景色最佳
- 地铁2号线苜蓿园站或下马坊站均可到达
- 每年2-3月梅花节、11月银杏季为最佳游览时间

## 2. 夫子庙-秦淮河

### 📍位置
南京市秦淮区贡院西街53号，位于南京城南秦淮河畔。

### 🏷特色
夫子庙-秦淮河风光带是南京最具代表性的历史文化景区，被誉为"十里秦淮、六朝金粉"。夫子庙即孔庙，始建于东晋，是中国四大文庙之一，秦淮河畔自古便是文人墨客荟萃之地。景区以秦淮河为轴线，串联起夫子庙大成殿、江南贡院、乌衣巷、瞻园、老门东等历史遗迹。江南贡院是中国古代最大的科举考场，鼎盛时期拥有号舍20644间。夜间乘画舫夜游秦淮河，两岸灯火辉煌，桨声灯影中感受"烟笼寒水月笼沙"的诗意。老门东历史文化街区保留了明清建筑风貌，汇聚了南京传统手工艺和特色小吃，是体验南京烟火气的最佳去处。

### 🗺推荐旅游路线

**一日游路线：**
夫子庙大成殿 → 江南贡院 → 乌衣巷 → 午餐（老门东小吃街）→ 瞻园 → 老门东 → 秦淮河画舫夜游

**深度游路线：**
Day 1：夫子庙大成殿 → 江南贡院 → 乌衣巷 → 秦淮河夜游
Day 2：老门东 → 瞻园 → 甘熙宅第 → 中华门瓮城 → 雨花台
Day 3：南京博物院 → 明故宫遗址 → 午朝门公园 → 朝天宫

### 🍜附近饮食

| 美食 | 说明 | 推荐店铺 |
|------|------|---------|
| 秦淮八绝 | 八套南京经典小吃点心组合 | 蒋有记、莲湖糕团店、奇芳阁 |
| 鸭血粉丝汤 | 南京地标小吃，汤鲜味美 | 回味鸭血粉丝汤 |
| 桂花盐水鸭 | 皮薄肉嫩，桂花飘香 | 韩复兴 |

- 景区全天开放，内部景点有具体开放时间

- 地铁1号线三山街站或3号线夫子庙站均可到达

## 3. 苏州园林（拙政园-留园-虎丘）

### 📍位置
- 拙政园：苏州市姑苏区东北街178号
- 留园：苏州市姑苏区留园路338号
- 虎丘：苏州市姑苏区山门内8号

### 🏷特色
苏州园林是中国古典园林的巅峰代表，被联合国教科文组织列为世界文化遗产。拙政园是苏州最大的古典园林，始建于明正德年间，以水为中心，亭台楼阁错落有致，分为东、中、西三部分，中园是全园精华。留园与北京颐和园、承德避暑山庄、苏州拙政园并称中国四大名园，以建筑空间处理艺术著称，长廊、漏窗、洞门巧妙分隔空间。虎丘有"吴中第一名胜"之誉，苏东坡曾言"到苏州不游虎丘，乃憾事也"。虎丘塔是世界第二斜塔，剑池传说吴王阖闾葬于池下。三处园林各具特色，共同构成了苏州园林艺术的完整画卷。

### 🗺推荐旅游路线

**一日游路线：**
拙政园（2小时）→ 苏州博物馆（午餐）→ 虎丘（1.5小时）→ 山塘街（晚餐+夜游）

**深度游路线：**
Day 1：拙政园 → 苏州博物馆 → 平江路 → 观前街
Day 2：留园 → 虎丘 → 山塘街夜游
Day 3：狮子林 → 网师园（夜花园）→ 沧浪亭 → 盘门景区
Day 4：周庄/同里古镇一日游

### 🍜附近饮食

| 美食 | 说明 | 推荐店铺 |
|------|------|---------|
| 苏式汤面 | 面汤鲜美，浇头丰富（焖肉、爆鱼、鳝丝） | 同得兴、裕兴记 |
| 松鼠桂鱼 | 苏帮菜代表，外酥里嫩，酸甜可口 | 松鹤楼、得月楼 |
| 蟹壳黄 | 苏州传统烧饼，酥脆咸香 | 万福兴 |

### 💡Tips

- 三处园林均需提前网上预约购票

- 地铁4号线北寺塔站可达拙政园附近

## 4. 周庄古镇

### 📍位置
苏州市昆山市周庄镇全福路，位于苏州城区东南约38公里。

### 🏷特色
周庄是中国第一水乡，有着900余年历史，因画家陈逸飞的双桥油画而闻名世界。古镇四面环水，井字形河道将全镇划分为八个板块，14座古桥连接着各个区域。双桥（世德桥和永安桥）一横一纵、一圆一方，是周庄的标志性景观。沈厅是江南巨富沈万三后裔所建的七进宅院，规模宏大；张厅则是典型的"轿从前门进，船从家中过"的明代建筑。夜幕降临后的周庄更是迷人，红灯笼倒映在水中，橹声咿呀，仿佛穿越回千年前的江南水乡。《四季周庄》实景演出以水乡为舞台，生动展现江南民俗风情。

### 🗺推荐旅游路线

**一日游路线：**
入口 → 双桥 → 沈厅 → 张厅 → 全福长桥 → 午餐（万三蹄）→ 迷楼 → 南湖秋月园 → 古镇水巷游船 → 夜游周庄

**深度游路线：**
Day 1：周庄古镇全天深度游（含夜游+实景演出）
Day 2：锦溪古镇 → 千灯古镇
Day 3：甪直古镇 → 苏州工业园区金鸡湖

### 🍜附近饮食

| 美食 | 说明 | 推荐店铺 |
|------|------|---------|
| 万三蹄 | 周庄招牌菜，皮色酱红，肥而不腻 | 沈厅酒家 |
| 阿婆菜 | 周庄传统腌菜，咸鲜开胃 | 古镇沿街小铺 |
| 青团 | 艾草汁糯米团，甜馅或咸馅 | 贞丰酥点 |
| 三味圆 | 周庄特色汤圆，鲜肉/豆沙/芝麻三味 | 古镇街边 |

### 💡Tips
- 建议住一晚，体验清晨无人的古镇和夜景
- 古镇门票100元，包含镇内所有景点
- 摄影爱好者建议清晨6-7点拍摄，游客少、光线好
- 周末和节假日人流量大，建议工作日前往
- 从苏州汽车站有直达班车，约1小时车程

## 5. 同里古镇

### 📍位置
苏州市吴江区同里镇中川南路1号，距苏州市区约18公里。

### 🏷特色
同里古镇始建于宋代，有着一千多年历史，是江南六大古镇之一。古镇被"川"字形的15条小河分隔成七个岛屿，49座古桥将各岛连为一体，真正实现了"小桥流水人家"的水乡意境。退思园是同里的灵魂，虽仅占地九亩八分，却集清代园林艺术之大成，被列为世界文化遗产。崇本堂和嘉荫堂的精美木雕令人叹为观止。三桥（太平桥、吉利桥、长庆桥）是同里人"走三桥"祈福的习俗所在。同里比周庄更安静、更原生态，镇上至今仍有大量原住民生活，清晨可见老人们在河边洗衣、午后有评弹声从茶馆飘出，是真正活着的江南古镇。

### 🗺推荐旅游路线

**一日游路线：**
退思园 → 崇本堂 → 嘉荫堂 → 三桥（走三桥）→ 午餐 → 南园茶社（品茶听评弹）→ 珍珠塔景区 → 罗星洲

**深度游路线：**
Day 1：同里古镇全天深度游
Day 2：黎里古镇 → 震泽古镇
Day 3：苏州城区园林游（拙政园+留园）

### 🍜附近饮食

| 美食 | 说明 | 推荐店铺 |
|------|------|---------|
| 状元蹄 | 同里名菜，猪蹄酥烂入味 | 同里状元蹄老店 |
| 闵饼 | 同里特色糕点，糯米制成 | 闵饼老店 |
| 袜底酥 | 薄如袜底的酥饼，椒盐味 | 同里糕团店 |
| 芡实粥 | 水乡养生粥品，软糯清香 | 古镇茶社 |

### 💡Tips
- 建议住宿一晚，体验古镇清晨和夜晚的宁静
- 退思园是必看景点，世界文化遗产
- "走三桥"是当地习俗，寓意太平吉利，不妨体验
- 古镇内有多家评弹茶馆，花20-30元可品茶听曲
- 从苏州火车站有直达公交，约45分钟

## 6. 鼋头渚

### 📍位置
无锡市滨湖区鼋渚路1号，位于太湖之滨、充山半岛前端。

### 🏷特色
鼋头渚是横卧太湖西北岸的一个半岛，因巨石突入湖中形如浮鼋翘首而得名，被誉为"太湖佳绝处、毕竟在鼋头"。景区面积达500多公顷，是太湖风景的精华所在。每年3-4月的樱花季是鼋头渚最壮观的时刻，3万多株樱花竞相绽放，长春桥畔樱花如云似霞，被誉为"中华第一赏樱胜地"。景区内有鼋头渚灯塔、横云山庄、藕花深处、太湖仙岛等众多景点。乘船前往太湖仙岛，可远眺太湖三万六千顷的浩渺烟波。郭沫若曾赞叹"太湖佳绝处，毕竟在鼋头"，使其名扬天下。

### 🗺推荐旅游路线

**一日游路线：**

**深度游路线：**
Day 1：鼋头渚全天游览
Day 2：蠡园 → 蠡湖大桥 → 无锡融创乐园
Day 3：灵山大佛 → 拈花湾小镇

## 7. 灵山大佛

### 📍位置
无锡市滨湖区马山灵山路1号，位于太湖之滨的小灵山南麓。

### 🏷特色
灵山大佛是世界上最高大的露天青铜释迦牟尼佛立像，通高88米，佛体79米，莲花座9米。大佛慈颜微笑，广视众生，右手"施无畏印"代表除却痛苦，左手"与愿印"代表给予快乐。景区以灵山大佛为核心，集佛教文化、自然山水、人文景观于一体。九龙灌浴大型音乐喷泉表演再现了佛祖降生时的祥瑞场景，气势恢宏。灵山梵宫被誉为"东方的卢浮宫"，内部金碧辉煌，穹顶壁画精美绝伦。五印坛城是藏传佛教风格建筑，可体验唐卡绘画、转经筒等藏传文化。景区内的拈花湾禅意小镇是近年新开发的禅意度假区，仿宋建筑风格，夜晚灯光秀美轮美奂。

## 9. 天目湖

### 📍位置
常州市溧阳市天目湖镇，距溧阳市区约8公里。

### 🏷特色
天目湖被誉为"江南明珠"，是长三角地区难得的天然山水景观。湖区内群山环抱、湖水清冽，自然植被覆盖率高达95%以上。天目湖山水园是核心景区，湖中有龙兴岛、绣球岛、鸟岛等多个岛屿。乘游船穿行于碧波之间，远山如黛，近水含烟，令人心旷神怡。龙兴岛上有蝴蝶谷和精灵国，适合亲子游。南山竹海景区位于天目湖南部，万亩竹海连绵起伏，乘坐小火车穿越竹林别有趣味。御水温泉是国内十大温泉之一，富含多种矿物质。天目湖砂锅鱼头是当地一绝，用天目湖水和大灰鲢鱼头熬制而成，汤色乳白，鲜美无比。

### 🗺推荐旅游路线

**一日游路线：**

Day 3：曹山慢城 → 溧阳1号公路 → 溧阳博物馆

## 11. 瘦西湖

### 📍位置
扬州市邗江区大虹桥路28号，位于扬州市西北部。

### 🏷特色
瘦西湖是扬州的一张金名片，原名保障湖，因湖面狭长、清秀婉约而得名"瘦西湖"。景区全长约4.3公里，面积100多公顷，融南方之秀与北方之雄于一体。五亭桥是瘦西湖的标志性建筑，桥上五亭如莲花绽放，桥下十五个桥洞在月圆之夜各映一轮明月，被誉为"中国最美的桥"。二十四桥因杜牧"二十四桥明月夜，玉人何处教吹箫"而闻名。白塔是仿北京北海白塔而建，与五亭桥相映成趣。每年4月的"烟花三月"旅游节期间，湖畔垂柳依依、桃花灼灼，再现"烟花三月下扬州"的诗意画卷。春季是瘦西湖最美的季节，也是扬州旅游的黄金时期。

## 12. 金山·焦山·北固山

### 📍位置
- 金山：镇江市润州区金山路62号

镇江三山是长江与京杭大运河交汇处的三座名山，自古便是军事要塞和文人墨客题咏之地。金山以"白娘子水漫金山"的传说闻名天下，金山寺依山而建，山寺一体，形成"寺裹山"的独特景观，法海洞、白龙洞等传说遗迹众多。焦山是长江中唯一的四面环水的游览岛，山中焦山碑林收藏了从南朝至清代的碑刻460余方，其中最珍贵的是"大字之祖"《瘗鹤铭》。北固山以"天下第一江山"闻名，刘备甘露寺招亲的故事就发生在此，北固楼登高远眺，大江东去的壮阔尽收眼底。三山各具特色，共同展示了镇江"城市山林"的独特魅力。

## 13. 濠河

### 📍位置
南通市崇川区濠河风景区，环绕南通老城区。

### 🏷特色
濠河是国内保存最完整的古护城河之一，全长约10公里，水面面积近70公顷，呈葫芦形环绕南通老城区，形成了"水抱城、城拥水、城水一体"的独特格局。沿河两岸分布着天宁寺、光孝塔、文峰塔、南通博物苑（中国第一座公共博物馆）、沈寿艺术馆等众多历史文化景点。濠河之美在于其原生态的生活气息——晨练的老人、垂钓的市民、戏水的孩童，城市与河流和谐共生。夜晚乘船游濠河，两岸灯火与倒影交相辉映，沿途可观赏各种灯光主题秀。南通博物苑由清末状元张謇创办，开中国博物馆事业之先河，馆内珍品众多。

## 14. 溱湖

### 📍位置
泰州市姜堰区溱湖大道1号，位于泰州北部。

### 🏷特色
溱湖国家湿地公园是国家5A级旅游景区，总面积约26平方公里，其中水域面积占近一半。溱湖是典型的淡水湖泊湿地，湖中岛屿星罗棋布，湖畔芦苇丛生，水禽翔集。景区以"水、湿地、生态"为主题，拥有湿地科普馆、湿地探险乐园、水禽园等景点。每年清明节期间举办的溱潼会船节是国家级非物质文化遗产，数百条船只在湖面上竞渡，场面壮观。景区内可乘摇橹船穿行于芦苇荡中，观赏各种水鸟和湿地植物。鹿鸣岛上散养着数十头梅花鹿，可近距离互动。溱湖八鲜（溱湖簖蟹、溱湖甲鱼、溱湖银鱼等）是当地特色美食。

### 🗺推荐旅游路线

**一日游路线：**
景区入口 → 湿地科普馆 → 乘船至湖心岛 → 鹿鸣岛 → 午餐（溱湖八鲜）→ 湿地探险乐园 → 水禽园 → 古农具展厅

## 23. 中华麋鹿园

### 📍位置
盐城市大丰区草庙镇，位于黄海之滨的大丰麋鹿国家级自然保护区。

### 🏷特色
中华麋鹿园是世界上最大的麋鹿自然保护区，占地面积约780公顷，拥有世界上最大的野生麋鹿种群，数量超过7000头。麋鹿因其"角似鹿、面似马、蹄似牛、尾似驴"而被称为"四不像"，是中国特有的珍稀动物。景区分为圈养区和野生放养区，圈养区可近距离观赏麋鹿，野生放养区则可看到麋鹿在自然环境中自由奔跑的壮观场面。保护区内还栖息着丹顶鹤、白枕鹤、天鹅等多种珍稀鸟类，是观鸟爱好者的天堂。每年6-8月是麋鹿的产仔期，可以看到可爱的麋鹿宝宝。景区内的麋鹿博物馆详细介绍了麋鹿从灭绝到重引进的传奇历史。

### 🗺推荐旅游路线

**一日游路线：**
景区入口 → 麋鹿博物馆 → 圈养观赏区 → 观鹿台（远眺野生麋鹿群）→ 午餐 → 野生放养区（乘坐观光车）→ 湿地栈道 → 观鸟台

**深度游路线：**
Day 1：中华麋鹿园全天游
Day 2：盐城丹顶鹤湿地保护区 → 盐城新四军纪念馆
Day 3：大丰港 → 黄海森林公园

## 24. 周恩来故里

### 📍位置
淮安市淮安区驸马巷7号，位于淮安古城中心。

### 🏷特色
周恩来故里是为纪念伟大的无产阶级革命家周恩来总理而建立的红色旅游景区，由周恩来纪念馆、周恩来故居、驸马巷历史街区和河下古镇四大片区组成。周恩来纪念馆由齐康院士设计，造型庄重典雅，馆内陈列了大量珍贵的历史照片、文献和实物，全面展示了周恩来总理波澜壮阔的一生和崇高的精神风范。周恩来故居是一座典型的苏北民居，周恩来在此度过了12年的童年时光。驸马巷保留了明清时期的街巷格局。河下古镇有2500多年历史，是淮安历史文化的核心区域，《西游记》作者吴承恩就出生于此。古镇内文楼汤包、平桥豆腐等淮扬美食令人垂涎。

**一日游路线：**

## 25. 洪泽湖湿地

### 📍位置
宿迁市泗洪县城头乡，位于洪泽湖西岸。

### 🏷特色
洪泽湖湿地公园是华东地区面积最大的淡水湿地，总面积约20平方公里，拥有万亩芦苇荡、千亩荷花荡和丰富的鸟类资源。景区以湿地生态为核心，分为湿地科普区、荷花观赏区、芦苇迷宫区和鸟类观赏区。夏季（7-8月）千亩荷花竞相绽放，接天莲叶无穷碧，映日荷花别样红，可乘船穿行于荷花丛中。秋季芦花飘飞，如梦如幻。保护区内栖息着白鹭、苍鹭、鸬鹚、野鸭等200多种鸟类，是观鸟和摄影的绝佳去处。景区还设有湿地博物馆、鸟类科普馆、丛林穿越等体验项目，是亲近自然、亲子游的理想目的地。

### 🗺推荐旅游路线

**一日游路线：**

**深度游路线：**
Day 1：洪泽湖湿地全天游
Day 2：洪泽湖大堤 → 老子山温泉
Day 3：项王故里 → 三台山森林公园

| 洪泽湖大闸蟹 | 洪泽湖特产，秋季肥美 | 湖畔蟹庄 |
| 泗洪空心挂面 | 泗洪传统面食，面条中空 | 泗洪特产店 |

## 26. 连岛

### 📍位置
连云港市连云区连岛镇，是江苏省最大的海岛。

### 🏷特色
连岛是江苏省最大的基岩海岛，面积约7.6平方公里，通过拦海大堤与大陆相连。连岛拥有江苏最好的海滨浴场——大沙湾海滨浴场，沙滩金黄细软，海水清澈湛蓝，是江苏唯一优质的海滨度假地。苏马湾是连岛另一处天然海湾，比大沙湾更幽静，沙滩更细腻，被誉为"江苏的北戴河"。连岛海滨栈道全长约5公里，沿海岸线蜿蜒而行，一边是苍翠的山林，一边是蔚蓝的大海，风景壮美。环岛公路可骑行或步行，沿途可观赏奇石怪礁、海蚀地貌。海天一色观景台是拍摄日出的最佳位置，清晨可见太阳从海面升起，金光万道。

### 🗺推荐旅游路线

**一日游路线：**
连岛入口 → 大沙湾海滨浴场（戏水）→ 海滨栈道 → 午餐（海鲜）→ 苏马湾 → 海天一色观景台 → 日落

**深度游路线：**
Day 1：连岛全天游（含日出日落）
Day 2：花果山一日游
Day 3：连云港老街 → 海州古城 → 孔望山

| 连云港海鲜 | 黄海新鲜海产，梭子蟹、对虾、黄鱼 | 连岛海鲜排档 |
| 赣榆煎饼 | 苏北传统面食，薄脆可口 | 岛上小摊 |
| 花果山凤鹅 | 连云港特色腌制鹅肉 | 连云港特产店 |

### 💡Tips
