## 1. 庐山

**📍位置**：九江市境内，距九江市区约30公里

**🏷特色**：世界文化景观遗产，以"匡庐奇秀甲天下"闻名，集雄、奇、险、秀于一体，是中国山水诗的摇篮，拥有丰富的历史遗迹和万国别墅群。

**🗺推荐旅游路线**

- **一日游（精华线）**：牯岭镇 → 花径 → 如琴湖 → 锦绣谷 → 仙人洞 → 险峰 → 美庐 → 庐山会议旧址 → 含鄱口（远眺鄱阳湖）→ 五老峰（简略）→ 返程
- **三日游（全景深度）**：
  - Day1：牯岭镇 → 花径 → 如琴湖 → 锦绣谷 → 仙人洞 → 险峰 → 御碑亭 → 美庐 → 庐山会议旧址
  - Day2：含鄱口（日出）→ 五老峰（登顶）→ 三叠泉（庐山第一瀑）→ 返回牯岭
  - Day3：白鹿洞书院（山下）→ 秀峰瀑布（李白"飞流直下三千尺"）→ 东林大佛 → 返程

**🍜附近饮食**

| 美食名称 | 特色介绍 | 推荐地点 |
|---------|---------|---------|
| 庐山石鸡 | 棘胸蛙红烧或清蒸，肉质细嫩 | 牯岭镇餐馆 |
| 庐山石鱼 | 庐山溪流小鱼，炒蛋或煮汤，鲜香 | 牯岭镇 |
| 庐山石耳 | 崖壁木耳炖鸡汤，滑嫩滋补 | 牯岭镇 |
| 萝卜饼 | 九江特色，萝卜丝馅油煎饼，外酥内鲜 | 九江市区 |

**💡Tips**
- 庐山山上（牯岭镇）海拔约1100米，夏季平均22°C，是著名避暑胜地
- 三叠泉需下1400级台阶才能到达瀑布底，返程爬升体力消耗大
- 自驾上山需提前预约车辆（旺季限流），也可在山下换乘观光车
- 含鄱口看日出需凌晨4-5点出发，提前查询天气
- 牯岭镇住宿、餐饮设施完善，可作为大本营

## 2. 三清山

**📍位置**：上饶市玉山县与德兴市交界处，距玉山县城约50公里

**🏷特色**：2008年列入世界自然遗产名录，以花岗岩峰林地貌著称，拥有"西太平洋最美花岗岩"之誉，东方女神和巨蟒出山是标志性景观。

**🗺推荐旅游路线**

- **一日游（南清园环线）**：南部索道上站 → 南清园景区（东方女神 → 巨蟒出山 → 一线天 → 玉女开怀）→ 西海岸栈道（高空栈道观云海）→ 阳光海岸栈道 → 南部索道下站
- **两日游（全景环线）**：
  - Day1：南部索道上 → 南清园 → 西海岸栈道（观日落）→ 住山顶
  - Day2：玉京峰（日出）→ 阳光海岸 → 三清宫 → 万寿园 → 索道下山

**🍜附近饮食**

| 美食名称 | 特色介绍 | 推荐地点 |
|---------|---------|---------|
| 三清山野菜 | 山区时令野蔬，蕨菜、马兰头等，清新鲜美 | 山下农家乐 |
| 上饶鸡腿 | 腌制鸡腿肉串烤，香辣可口，上饶街头名小吃 | 上饶火车站/玉山 |
| 玉山烤饼 | 薄饼内夹肉末和梅干菜，炭火烤制 | 玉山县城 |
| 铅山烫粉 | 滚汤中烫熟的米粉，配肉丝和鲜汤 | 上饶市区 |

**💡Tips**
- 三清山索道分南部和东部两条，南部索道最常用
- 西海岸栈道是世界上海拔最高的高空栈道，恐高者需心理准备
- 雨后云海概率大，建议关注天气预报选择时机
- 山顶住宿有限且价格高，建议提前预订或选择一日游
- 建议购买往返索道票，山上步行全程约需8-10小时

## 3. 井冈山

**📍位置**：吉安市井冈山市，距吉安市区约110公里

**🏷特色**：中国革命的摇篮，1927年毛泽东在此创建了中国第一个农村革命根据地，以红色革命遗址和优美的自然风光并存著称。

**🗺推荐旅游路线**

- **一日游（红色经典）**：井冈山革命博物馆 → 井冈山革命烈士陵园 → 茨坪毛泽东旧居 → 黄洋界（著名的黄洋界保卫战遗址）→ 龙潭瀑布群 → 返程
- **两日游（红色+绿色）**：
  - Day1：革命博物馆 → 烈士陵园 → 茨坪旧居 → 大井朱毛旧居 → 水口彩虹瀑布
  - Day2：黄洋界 → 龙潭瀑布群 → 百竹园 → 笔架山杜鹃花长廊（4-5月）→ 返程

**🍜附近饮食**

| 美食名称 | 特色介绍 | 推荐地点 |
|---------|---------|---------|
| 红米饭南瓜汤 | 红军时期的传统饮食，红色文化的象征 | 茨坪红军餐厅 |
| 井冈山烟笋 | 烟熏竹笋炒腊肉，山野风味浓郁 | 茨坪餐馆 |
| 永新狗肉 | 永新县特色，红烧或白切，肉质鲜美 | 吉安餐馆 |
| 泰和乌鸡汤 | 泰和特产乌鸡炖汤，滋补佳品 | 井冈山/吉安 |

**💡Tips**
- 井冈山革命博物馆免费参观，展陈丰富，建议安排2小时
- 黄洋界地势险要，"黄洋界上炮声隆"的诗句即出于此
- 龙潭瀑布群共五潭五瀑，其中碧玉潭落差最大（67米）
- 4-5月笔架山杜鹃花海漫山遍野，极为壮观
- 景区观光车通票50元，覆盖所有景点，建议购买

## 4. 龙虎山

**📍位置**：鹰潭市西南约20公里

**🏷特色**：中国道教发祥地，东汉张道陵在此炼丹修道，丹霞地貌与道教文化完美融合，泸溪河竹筏漂流观丹霞和悬棺表演是最大亮点。

**🗺推荐旅游路线**

- **一日游**：游客中心 → 泸溪河竹筏漂流（约1.5小时，两岸丹霞奇观）→ 仙水岩（观看升棺表演，古越族悬棺）→ 象鼻山 → 天师府（道教祖庭）→ 上清古镇 → 返程
- **两日游**：
  - Day1：泸溪河竹筏漂流 → 仙水岩悬棺表演 → 龙虎山地质博物馆 → 住景区
  - Day2：天师府 → 上清古镇 → 大上清宫遗址 → 正一观 → 返程

**🍜附近饮食**

| 美食名称 | 特色介绍 | 推荐地点 |
|---------|---------|---------|
| 鹰潭牛肉粉 | 汤底醇厚的牛肉粉，早餐首选 | 鹰潭市区 |
| 天师板栗烧鸡 | 以天师命名的道家名菜，板栗与鸡同烧 | 上清古镇 |
| 上清豆腐 | 龙虎山上清镇的手工豆腐，细嫩滑口 | 上清古镇 |
| 贵溪捺菜 | 腌制的芥菜，酸辣脆爽，开胃下饭 | 鹰潭餐馆 |

**💡Tips**
- 泸溪河竹筏漂流建议上午前往，光线适合拍照
- 升棺表演在固定时间进行（通常10:00、12:00、14:00），需提前确认
- 天师府是中国道教正一派的祖庭，张天师世袭至今已60余代
- 上清古镇保存了大量明清建筑，可顺道游览
- 景区内观光车免费乘坐，覆盖主要景点

## 5. 婺源江湾

**📍位置**：上饶市婺源县江湾镇，距婺源县城约28公里

**🏷特色**：婺源被誉为"中国最美乡村"，江湾是婺源最大的古村落之一，保留了完整的徽派建筑群，三月油菜花和秋季晒秋是两大名片。

**🗺推荐旅游路线**

- **一日游（东线经典）**：江湾古镇（萧江宗祠、永思堂）→ 篁岭（晒秋人家）→ 李坑（小桥流水人家）→ 返程
- **两日游（东线+北线）**：
  - Day1：江湾 → 篁岭（含索道，晒秋美景）→ 住婺源县城或篁岭民宿
  - Day2：李坑 → 汪口 → 晓起 → 思溪延村 → 彩虹桥 → 返程
- **春季特别线**：江岭梯田油菜花（3月中旬-4月上旬，日出晨雾）→ 篁岭 → 江湾

**🍜附近饮食**

| 美食名称 | 特色介绍 | 推荐地点 |
|---------|---------|---------|
| 荷包红鲤鱼 | 婺源特产，红鲤鱼清蒸，肉质细嫩味鲜 | 婺源餐馆 |
| 粉蒸肉 | 米粉裹五花肉蒸制，肥而不腻 | 江湾农家乐 |
| 气糕 | 米浆发酵后蒸制的发糕，松软微甜 | 婺源街头 |
| 清明粿 | 鼠曲草汁和糯米制成，清明时令美食 | 婺源小摊 |

**💡Tips**
- 婺源通票210元（5天有效），涵盖江湾、李坑、汪口等12个景点，性价比高
- 篁岭单独售票（145元含索道），不在通票范围内
- 三月油菜花期游客极多，江岭观景台建议清晨5点前往
- 婺源各景点较分散，自驾最方便，也可包车（约300-400元/天）
- 晒秋景观在篁岭最佳，秋季（9-11月）色彩最丰富

## 6. 滕王阁

**📍位置**：南昌市东湖区沿江路，赣江东岸

**🏷特色**：江南三大名楼之一，因王勃《滕王阁序》中"落霞与孤鹜齐飞，秋水共长天一色"而名扬千古，现楼为1989年重建的第29代滕王阁。

**🗺推荐旅游路线**

- **半日游**：景区入口 → 滕王阁主楼（逐层参观，顶层观赣江风光）→ 仿古街 → 赣江边漫步 → 返程
- **一日游串联**：滕王阁 → 八一广场 → 绳金塔美食街 → 秋水广场（晚上音乐喷泉）→ 南昌之星摩天轮

**🍜附近饮食**

| 美食名称 | 特色介绍 | 推荐地点 |
|---------|---------|---------|
| 南昌拌粉 | 米粉拌酱油、辣椒、花生米，南昌人的早餐灵魂 | 绳金塔美食街 |
| 瓦罐汤 | 小瓦罐慢火煨制，品种丰富，鲜美醇厚 | 绳金塔/南昌街头 |
| 白糖糕 | 传统米糕，外脆内软，甜而不腻 | 南昌老字号 |
| 藜蒿炒腊肉 | 南昌特色家常菜，藜蒿清香配腊肉咸鲜 | 南昌餐馆 |

**💡Tips**
- 滕王阁夜景灯光璀璨，建议傍晚前往，同时感受日景和夜景
- 阁内展陈有历代滕王阁模型和《滕王阁序》碑刻
- 秋季（9-11月）傍晚赣江边可观赏"落霞与孤鹜齐飞"的意境
- 距南昌火车站约10分钟车程，交通便利
- 南昌拌粉和瓦罐汤是必吃组合，推荐绳金塔美食街

## 7. 明月山

**📍位置**：宜春市袁州区西南部，距宜春市区约30公里

**🏷特色**：以奇峰险壑、温泉飞瀑和禅宗文化著称，山顶拥有万余亩高山草甸，月亮文化是景区主题，被誉为"月亮之都"。

**🗺推荐旅游路线**

- **一日游**：景区大门 → 云谷飞瀑 → 青云栈道（悬崖栈道）→ 月亮湖 → 乌云崖 → 星月洞 → 索道下山 → 天沐温泉（山脚泡温泉）
- **两日游**：
  - Day1：明月山登山全程 → 住山顶帐篷或酒店
  - Day2：观日出 → 高山草甸徒步 → 下山 → 温泉小镇泡汤

**🍜附近饮食**

| 美食名称 | 特色介绍 | 推荐地点 |
|---------|---------|---------|
| 宜春扎粉 | 手工米粉扎成团，炒制或煮汤，口感筋道 | 宜春市区 |
| 丰城冻米糖 | 糯米爆米花加糖压制成块，酥脆甜香 | 宜春特产店 |
| 慈化鸡 | 宜春特色烧鸡，酱香浓郁 | 宜春餐馆 |
| 万载百合粉 | 万载特产百合制成的冲泡粉，清甜养生 | 明月山土特产店 |

**💡Tips**
- 明月山索道单程80元，建议至少单程乘坐，节省体力
- 青云栈道全长约3100米，悬挂在海拔约1500米的绝壁上，壮美惊险
- 山脚的天沐温泉、维景温泉等适合下山后放松
- 明月山与温汤镇可串联，温汤镇富硒温泉远近闻名
- 中秋节期间景区有月亮文化节活动

## 8. 景德镇古窑民俗博览区

**📍位置**：景德镇市瓷都大道古窑路1号

**🏷特色**：中国唯一以陶瓷文化为主题的5A景区，展示了从宋代到清代的古窑炉群，可现场观看传统手工制瓷工艺，是了解中国陶瓷文化的最佳窗口。

**🗺推荐旅游路线**

- **一日游**：景区大门 → 历代古窑展示区（宋代龙窑、元代馒头窑、明代葫芦窑、清代镇窑）→ 手工制瓷作坊（拉坯、画坯体验）→ 瓷音水榭 → 明清古建筑群 → 陶瓷民俗展示 → 返程
- **串联游**：古窑景区 → 景德镇中国陶瓷博物馆（免费）→ 陶溪川文创街区（夜市）→ 三宝国际陶艺村

**🍜附近饮食**

| 美食名称 | 特色介绍 | 推荐地点 |
|---------|---------|---------|
| 碱水粑 | 大米加碱水制成的糕片，炒制后外焦里嫩 | 景德镇街头 |
| 冷粉 | 景德镇特色拌粉，辣椒酱拌凉粉，爽辣开胃 | 景德镇早餐店 |
| 饺子粑 | 辣椒豆腐或韭菜鸡蛋馅的蒸饺，个头小 | 景德镇小摊 |
| 瓷泥煨鸡 | 鸡裹瓷泥烤制，肉质酥烂，景德镇独特做法 | 景德镇餐馆 |

**💡Tips**
- 古窑景区内可观看非遗传承人现场演示拉坯、利坯、画坯等72道工序
- 清代镇窑（蛋形窑）是中国最大的柴窑，每年会点火复烧一次
- 陶溪川是景德镇最热门的夜市，每周五周六有创意集市
- 景德镇陶瓷大世界可淘到性价比高的瓷器
- 建议安排景德镇至少两天，除了古窑还有三宝村、御窑厂等值得探访

## 9. 瑞金共和国摇篮

**📍位置**：赣州市瑞金市，距赣州市区约120公里

**🏷特色**：中华苏维埃共和国临时中央政府诞生地，被誉为"共和国摇篮"，叶坪和沙洲坝两大景区保存了大量红色革命旧址。

**🗺推荐旅游路线**

- **一日游**：叶坪革命旧址群（第一次全国苏维埃代表大会会址、红军烈士纪念塔、博生堡）→ 午餐 → 沙洲坝（红井、中华苏维埃临时中央政府大礼堂、毛泽东旧居）→ 二苏大景区 → 返程
- **半日游**：叶坪旧址群 → 沙洲坝红井 → 返程

**🍜附近饮食**

| 美食名称 | 特色介绍 | 推荐地点 |
|---------|---------|---------|
| 瑞金牛肉汤 | 现宰牛肉煮汤，肉嫩汤鲜，配辣椒酱 | 瑞金市区 |
| 赣南脐橙 | 全国著名水果，汁多味甜（11-12月采摘季） | 瑞金果园 |
| 兴国米粉鱼 | 米粉裹鱼片蒸制，鲜嫩滑口 | 瑞金/赣州餐馆 |
| 瑞金鱼丸 | 手工鱼丸，弹牙鲜美 | 瑞金餐馆 |

**💡Tips**
- 叶坪和沙洲坝相距约6公里，建议安排一整天
- "吃水不忘挖井人"的红井就在沙洲坝，可现场品尝井水
- 景区免费参观，但需预约
- 瑞金是客家文化区域，可体验客家围屋和民俗
- 建议结合赣州古城墙、通天岩等组成赣南红色文化之旅

## 10. 大觉山

**📍位置**：抚州市资溪县境内，距资溪县城约15公里

**🏷特色**：以原始森林和峡谷漂流著称，大觉山漂流全长约3.6公里，落差近200米，被誉为"亚洲第一漂"，同时拥有壮观的高山湖泊和丹霞地貌。

**🗺推荐旅游路线**

- **一日游（漂流为主）**：景区入口 → 大觉山索道上站 → 大觉岩寺（悬崖上的千年古刹）→ 索道下山 → 大觉山漂流（约1.5小时）→ 返程
- **非漂流一日游**：索道上山 → 大觉岩寺 → 天鹅湖 → 松涛听泉 → 悬空栈道 → 索道下山

**🍜附近饮食**

| 美食名称 | 特色介绍 | 推荐地点 |
|---------|---------|---------|
| 资溪面包 | 资溪人在全国开面包店，本地面包品质极佳 | 资溪县城 |
| 黎川灌芯糖 | 传统拉丝糖，外脆内软，花生芝麻馅 | 抚州特产店 |
| 资溪畲族乌饭 | 畲族传统，乌树叶汁浸泡糯米蒸制 | 资溪农家乐 |
| 南丰蜜橘 | 南丰县特产，皮薄汁甜（10-12月） | 抚州水果店 |

**💡Tips**
- 大觉山漂流为季节性项目，通常5-10月开放，需关注天气
- 漂流会全身湿透，务必带好防水袋保护手机和相机
- 大觉岩寺位于海拔1100米的悬崖岩洞中，建寺历史超千年
- 资溪是"中国面包之乡"，全县有近2万人在全国各地开面包店
- 景区距抚州市区约2小时车程，建议自驾

## 11. 龟峰

**📍位置**：上饶市弋阳县境内，距弋阳县城约10公里

**🏷特色**：丹霞地貌景区，因山石形似巨龟而得名，有"无山不龟、无石不龟"之说，是"中国丹霞"世界自然遗产的组成部分。

**🗺推荐旅游路线**

- **一日游**：景区大门 → 龟峰湖 → 老人峰 → 锁春洞 → 南岩寺（丹霞岩洞中的千年古寺）→ 三叠龟 → 展旗峰 → 龟背石 → 返回
- **深度游**：龟峰 + 南岩寺石窟（中国最大的丹霞石窟群）

**🍜附近饮食**

| 美食名称 | 特色介绍 | 推荐地点 |
|---------|---------|---------|
| 弋阳年糕 | 不粘不腻的年糕，三蒸两百锤制作 | 弋阳县城 |
| 弋阳扣肉 | 五花肉配梅菜蒸制，肥而不腻 | 弋阳餐馆 |
| 横峰葛粉 | 横峰县特产葛根粉，清热养生 | 上饶特产店 |
| 铅山灯盏粿 | 米浆做皮、萝卜丝做馅的蒸饺 | 铅山/弋阳 |

**💡Tips**
- 龟峰是典型的丹霞地貌科普基地，建议请导游讲解
- 南岩寺始建于晋代，依岩洞而建，规模宏大
- 龟峰距三清山约1.5小时车程，可串联游览
- 景区不大，半天可游完，适合与弋阳其他景点搭配
- 弋阳年糕是当地必买特产，口感与普通年糕截然不同

## 12. 武功山

**📍位置**：萍乡市芦溪县境内，距萍乡市区约50公里

**🏷特色**：以十万亩高山草甸著称，是国内同纬度最大的高山草甸，被誉为"云中草原"，是徒步爱好者和露营者的天堂。

**🗺推荐旅游路线**

- **一日游（索道速览）**：石鼓寺 → 中庵索道上站 → 金顶索道上站 → 金顶（海拔1918米，武功山最高峰）→ 高山草甸 → 原路索道返回
- **两日游（徒步穿越·经典）**：
  - Day1：沈子村 → 铁蹄峰 → 金顶（约6-7小时徒步）→ 金顶扎营观日落
  - Day2：金顶日出 → 吊马桩 → 绝望坡 → 好汉坡 → 发云界 → 龙山村（约8小时）

**🍜附近饮食**

| 美食名称 | 特色介绍 | 推荐地点 |
|---------|---------|---------|
| 萍乡小炒肉 | 萍乡名菜，辣椒炒肉，香辣过瘾 | 萍乡市区 |
| 萍乡花果 | 以柚子皮、冬瓜等制成的蜜饯，甜香 | 萍乡特产店 |
| 莲花血鸭 | 莲花县特色，鸭血与鸭肉同炒，香辣鲜美 | 萍乡/莲花县 |
| 艾米果 | 艾草汁和糯米制成的绿色糕点 | 萍乡街头 |

**💡Tips**
- 徒步穿越武功山是户外圈经典路线，需有基本徒步经验
- 金顶帐篷露营可看星空和日出，但山顶风大温低，需带厚衣服
- 高山草甸最美的季节是5-6月（绿草如茵）和9-10月（金黄一片）
- 索道分两段：中庵索道（上行65元）+ 金顶索道（上行45元）
- 周末和节假日金顶人满为患，建议工作日前往

## 13. 庐山西海

**📍位置**：九江市永修县和武宁县境内，距南昌约80公里

**🏷特色**：原名柘林湖，千岛落珠的大型水库型湖泊风景区，湖中3亩以上岛屿近千个，被誉为"江西千岛湖"，以湖光山色和水上运动著称。

**🗺推荐旅游路线**

- **一日游**：司马码头登船 → 奇幻岛（西海之星玻璃塔）→ 女神岛 → 明珠岛 → 桑田岛 → 箬溪岛 → 返回码头
- **深度游**：游船 + 水上运动（摩托艇、水上飞人）+ 湖畔温泉度假

**🍜附近饮食**

| 美食名称 | 特色介绍 | 推荐地点 |
|---------|---------|---------|
| 永修东坡肉 | 以苏东坡命名，红烧五花肉，酥烂入味 | 永修县城 |
| 武宁竹笋 | 新鲜竹笋炒腊肉，山野风味 | 武宁农家乐 |
| 柘林湖鱼头 | 湖中大鱼头炖汤，汤白味鲜 | 湖畔鱼庄 |
| 修水哨子 | 修水县特色，类似饺子但个头大，馅料丰富 | 九江餐馆 |

**💡Tips**
- 司马码头是主要登船码头，班次密集
- 西海之星玻璃塔是俯瞰全景的制高点
- 水上运动项目丰富，适合团队和家庭游玩
- 湖区紫外线强，注意防晒
- 距南昌近，适合周末一日或两日游

## 14. 三百山

**📍位置**：赣州市安远县东南部，距安远县城约30公里

**🏷特色**：东江源头，粤港澳大湾区饮用水的源头所在，以原始森林、瀑布群和温泉著称，被誉为"东江源头第一山"。

**🗺推荐旅游路线**

- **一日游**：景区入口 → 福鳌塘（源头标志）→ 东江第一瀑 → 知音泉 → 天印奇松 → 三叠瑶池 → 返程
- **深度游**：三百山登山 + 虎岗温泉（山脚天然温泉）

**🍜附近饮食**

| 美食名称 | 特色介绍 | 推荐地点 |
|---------|---------|---------|
| 安远三鲜粉 | 猪肉、猪肝、猪肠配米粉煮汤，鲜美 | 安远县城 |
| 赣南小炒鱼 | 鱼片滑炒，鲜嫩爽口，赣南经典菜 | 赣州/安远 |
| 安远酿豆腐 | 客家名菜，豆腐中酿入肉馅 | 安远餐馆 |
| 艾叶粄 | 艾草和糯米制成的客家小吃 | 安远小摊 |

**💡Tips**
- 三百山是东江源头，生态环境极好，负氧离子含量极高
- 景区相对小众，游客不多，适合喜欢清静的旅行者
- 东江第一瀑落差约30米，雨季水量充沛时最为壮观
- 虎岗温泉为天然温泉，水质优良
- 建议自驾前往，景区周边公共交通不便

## 15. 篁岭

**📍位置**：上饶市婺源县江湾镇篁岭村，距婺源县城约38公里

**🏷特色**：挂在山崖上的古村落，以"晒秋"民俗闻名全国，家家户户在屋顶晾晒红辣椒、黄玉米等农作物，形成五彩斑斓的丰收画卷，是中国"最美乡村符号"。

**🗺推荐旅游路线**

- **一日游**：索道上山 → 天街（古村落主街）→ 晒工坊（近距离观赏晒秋）→ 摄影吧（最佳拍摄点）→ 五桂堂 → 水口雨林 → 垒心桥（高空玻璃桥）→ 索道下山
- **深度游**：篁岭全天 + 夜宿篁岭民宿（夜景灯光秀）

**🍜附近饮食**

| 美食名称 | 特色介绍 | 推荐地点 |
|---------|---------|---------|
| 篁岭晒秋宴 | 以晒秋食材为主题的地方特色宴席 | 天街餐厅 |
| 荷包红鲤鱼 | 婺源名菜，红鲤鱼清蒸，色如荷包 | 村内农家乐 |
| 婺源糊豆腐 | 豆腐碎与肉末同煮成糊状，鲜嫩滑口 | 天街小吃 |
| 汽糕 | 米浆发酵蒸制的发糕，松软微甜 | 村内小摊 |

**💡Tips**
- 篁岭索道是唯一的上山方式，门票已含索道
- 晒秋最佳观赏期为9-11月，但四季都有农作物晾晒
- 摄影吧是拍摄晒秋全景的最佳位置，建议上午前往（光线好）
- 篁岭夜景近年打造得不错，推荐住一晚
- 与婺源其他景点可购买通票，但篁岭需单独购票
