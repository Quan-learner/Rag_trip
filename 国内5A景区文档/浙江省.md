# 浙江省 5A级旅游景区攻略

## 省份简介

浙江省，简称"浙"，位于中国东南沿海、长江三角洲南翼，东临东海，南接福建，西连江西、安徽，北接上海、江苏。浙江是中国经济最活跃的省份之一，以民营经济发达著称，素有"鱼米之乡、丝绸之府、文物之邦"的美誉。省会杭州是七大古都之一，西湖、大运河等世界文化遗产举世闻名。浙江山海兼备，地形以丘陵山地为主，有"七山一水二分田"之说。从杭州西湖的淡妆浓抹到普陀山的海天佛国，从乌镇的小桥流水到雁荡山的奇峰怪石，浙江的旅游资源极为丰富多元。截至2024年，浙江共有22家国家5A级旅游景区，数量位居全国前列。

## 1. 杭州西湖

### 📍位置
杭州市西湖区，位于杭州市中心西部。

### 🏷特色
杭州西湖是中国最具文化内涵的湖泊，2011年被列入世界文化遗产名录。西湖十景（苏堤春晓、曲院风荷、平湖秋月、断桥残雪、花港观鱼、柳浪闻莺、三潭印月、双峰插云、雷峰夕照、南屏晚钟）自南宋以来便名闻天下。苏堤是苏东坡任杭州知州时主持修建的，堤上六桥烟柳，春日最为迷人。断桥因《白蛇传》中许仙与白娘子的传说而成为爱情圣地。三潭印月是人民币一元纸币背面的图案，中秋之夜可见"天上月一轮，湖中影成三"的奇景。西湖之美在于自然与人文的完美融合——每一处景观背后都有诗词歌赋和历史典故，"欲把西湖比西子，淡妆浓抹总相宜"至今仍是最传神的描写。

### 🗺推荐旅游路线

**一日游路线：**
断桥残雪 → 白堤 → 孤山 → 曲院风荷 → 苏堤春晓 → 花港观鱼 → 午餐 → 雷峰塔 → 三潭印月（乘船）→ 南宋御街（晚餐）

**深度游路线：**
Day 1：西湖环湖一日游（断桥→白堤→孤山→苏堤→花港→雷峰塔→三潭印月）
Day 2：灵隐寺 → 飞来峰 → 龙井问茶 → 九溪烟树 → 钱塘江大桥
Day 3：西溪湿地 → 河坊街 → 南宋御街 → 城隍阁（夜景）
Day 4：良渚古城遗址 → 良渚博物院

### 💡Tips
- 西湖免费开放，建议步行或骑行环湖
- 三潭印月需乘船前往，船票55元含上岛
- 建议清晨或傍晚游览，避开正午人流高峰

- 租一辆共享单车环湖是最便捷的方式
- 地铁1号线龙翔桥站距离湖滨最近

## 2. 普陀山

### 📍位置
舟山市普陀区普陀山镇，是舟山群岛中的一个小岛。

### 🏷特色
普陀山是中国四大佛教名山之一，是观世音菩萨的道场，被誉为"海天佛国"。全岛面积约12.5平方公里，最高峰佛顶山海拔291.2米。岛上寺庙林立，最著名的有普济禅寺、法雨禅寺、慧济禅寺三大寺，以及不肯去观音院、紫竹林、南海观音铜像等。南海观音铜像高33米，面朝大海，庄严慈祥，是普陀山的标志性景观。普陀山四面环海，自然风光秀丽，千步沙、百步沙是优质的海滨沙滩，梵音洞、潮音洞等海蚀地貌令人叹为观止。每逢农历二月十九、六月十九、九月十九（观音诞辰、成道、出家日），数万信众登岛朝拜，香火鼎盛。

### 🗺推荐旅游路线

**一日游路线：**
码头 → 短姑道头 → 南海观音铜像 → 紫竹林 → 不肯去观音院 → 午餐 → 普济禅寺 → 千步沙 → 法雨禅寺

**深度游路线：**
Day 1：码头 → 南海观音 → 紫竹林 → 普济禅寺 → 千步沙（日落）
Day 2：法雨禅寺 → 佛顶山（慧济禅寺）→ 梵音洞 → 潮音洞 → 百步沙
Day 3：洛迦山一日游（"不到洛迦山，等于半朝山"）

### 🍜附近饮食

| 美食 | 说明 | 推荐店铺 |
|------|------|---------|
| 普陀素斋 | 佛门素食，清淡精致 | 普济禅寺素斋、法雨禅寺素斋 |
| 舟山海鲜 | 舟山渔场新鲜海产，梭子蟹、带鱼 | 沈家门海鲜排档 |
| 观音饼 | 普陀山特色糕点，甜糯可口 | 岛上特产店 |

### 💡Tips
- 建议至少住一晚，体验清晨的寺庙和夜晚的海浪
- 旺季（春节、观音诞辰日）船票紧张，需提前预订
- 岛上交通以步行和景区巴士为主，无出租车
- 进入寺庙请注意着装得体，保持安静
- 从沈家门半升洞码头或朱家尖蜈蚣峙码头乘船，约15-30分钟
- 佛教朝拜者建议请香自带，岛上的香价格较高

## 3. 雁荡山

### 📍位置
温州市乐清市雁荡镇，位于温州市东北部。

### 🏷特色
雁荡山被誉为"东南第一山"，是世界地质公园，以奇峰怪石、飞瀑流泉、古洞石室著称。雁荡山形成于1.2亿年前的火山喷发，属于流纹质火山岩地貌。灵峰、灵岩、大龙湫被称为"雁荡三绝"。灵峰日景看奇石，夜景看剪影——同一座山峰白天看像合掌，夜晚看像情侣，堪称"移步换景"的典范。灵岩飞渡表演是雁荡山的传统绝技，表演者在200多米高的悬崖峭壁间飞渡，惊险刺激。大龙湫瀑布落差197米，是中国单级落差最大的瀑布之一，雨季时气势磅礴，旱季时如丝如缕，各有韵味。徐霞客曾三次到访雁荡山，留下大量游记。

### 🗺推荐旅游路线

**一日游路线：**
灵峰景区（白天看奇石）→ 灵岩景区（飞渡表演+小龙湫）→ 午餐 → 大龙湫瀑布 → 方洞景区（悬崖栈道）→ 灵峰夜景

**深度游路线：**
Day 1：灵峰（日景+夜景）→ 灵岩 → 大龙湫
Day 2：方洞栈道 → 雁湖景区 → 显胜门
Day 3：楠溪江一日游（石桅岩→狮子岩→丽水街）

| 雁荡山石斛 | 珍贵中药材，可炖汤泡茶 | 雁荡山特产店 |
| 乐清鱼丸 | 温州特色鱼丸，鲜美弹牙 | 乐清老城区 |

### 💡Tips

- 灵岩飞渡表演有固定时间，需提前确认

- 雁荡山景区之间有景区巴士连接
- 从温州南站有直达雁荡山的班车

## 4. 千岛湖

### 📍位置
杭州市淳安县，距杭州市区约150公里。

### 🏷特色
千岛湖是1959年新安江水电站蓄水后形成的人工湖，湖中散布着1078个岛屿，故名"千岛湖"。湖区面积约573平方公里，水质常年保持国家一级标准，被誉为"天下第一秀水"。梅峰揽胜是千岛湖的标志性景点，登上梅峰岛观景台，可俯瞰300余座岛屿星罗棋布的壮观景象。黄山尖是另一个绝佳观景点，从高处看湖中岛屿排列成"天下为公"四个大字，令人称奇。龙川湾是千岛湖最美的湿地，秋色尤为迷人。千岛湖的有机鱼头是当地一绝，用千岛湖水养殖的有机鱼，鱼头汤色白味鲜。近年来千岛湖绿道骑行成为热门活动，环湖骑行约140公里，沿途风景如画。

### 🗺推荐旅游路线

**一日游路线：**
千岛湖码头 → 梅峰揽胜岛 → 午餐（鱼头汤）→ 龙山岛（海瑞祠）→ 月光岛 → 黄山尖

**深度游路线：**
Day 1：千岛湖中心湖区（梅峰+龙山+月光岛）
Day 2：千岛湖东南湖区 → 龙川湾湿地
Day 3：千岛湖绿道骑行 → 千岛湖啤酒小镇 → 秀水街

### 🍜附近饮食

| 美食 | 说明 | 推荐店铺 |
|------|------|---------|
| 千岛湖鱼头 | 千岛湖有机鱼头汤，汤白味鲜 | 千岛湖鱼味馆 |
| 淳安土鸡煲 | 山区散养土鸡，鲜美滋淳 | 千岛湖农家餐馆 |
| 千岛湖啤酒 | 千岛湖水酿造，口感清爽 | 千岛湖啤酒小镇 |

### 💡Tips
- 建议至少安排两天，一天游湖一天骑行
- 旺季（7-8月、国庆）船票紧张，需提前预订
- 梅峰揽胜建议乘坐缆车上山，步行下山
- 千岛湖有机鱼头一定要在当地品尝
- 从杭州汽车西站有直达班车，约2.5小时

## 5. 乌镇

### 📍位置
嘉兴市桐乡市乌镇镇，位于浙北杭嘉湖平原。

### 🏷特色
乌镇是中国最具代表性的水乡古镇之一，有着1300多年的建镇史，是世界互联网大会永久会址。古镇分为东栅和西栅两个景区——东栅保留了原汁原味的水乡风貌，茅盾故居、立志书院、江南百床馆等景点展示了乌镇的历史文化；西栅经过精心修缮，集观光、休闲、度假于一体，夜景尤为迷人。乌镇的水阁是独特的建筑形式——房屋一部分延伸至水面上，形成"人家尽枕河"的江南水乡意境。清晨的乌镇最迷人，薄雾笼罩河面，老人们在河边洗衣，这才是真正的水乡生活。每年乌镇戏剧节期间，古镇变身戏剧舞台，国内外先锋戏剧在此上演，是文艺青年的盛宴。

### 🗺推荐旅游路线

**一日游路线：**
西栅入口 → 草木本色染坊 → 叙昌酱园 → 白莲塔 → 午餐 → 水上集市 → 乌镇邮局 → 昭明书院 → 西栅夜游

**深度游路线：**
Day 1：东栅全天游（茅盾故居→立志书院→江南百床馆→宏源泰染坊）
Day 2：西栅全天游（含夜游）
Day 3：乌村体验 → 木心美术馆 → 互联网大会永久会址

## 6. 西塘

### 📍位置
嘉兴市嘉善县西塘镇，距上海市区约90公里。

### 🏷特色
西塘是江南六大古镇之一，有着千年的历史。古镇以"桥多、弄多、廊棚多"著称，最独特的是沿河而建的廊棚（烟雨长廊），全长约1300多米，遮风挡雨，是江南古镇中绝无仅有的建筑形式。西塘的弄堂是另一大特色——石皮弄最窄处仅0.8米，两人侧身才能通过。古镇内至今仍有大量原住民生活，比起过度商业化的古镇，西塘更多了一份生活气息。清晨和傍晚是西塘最迷人的时刻——清晨薄雾中老人们在河边生炉子，傍晚夕阳下炊烟袅袅，这才是原汁原味的江南。西塘的夜生活也很丰富，酒吧街热闹非凡，是年轻人夜游的好去处。

### 🗺推荐旅游路线

**一日游路线：**
景区入口 → 烟雨长廊 → 送子来凤桥 → 石皮弄 → 午餐 → 西园 → 中国酒文化博物馆 → 永宁桥（日落）→ 夜游酒吧街

**深度游路线：**
Day 1：西塘全天游（含夜游）
Day 2：嘉善巧克力甜蜜小镇 → 丁栅水乡
Day 3：乌镇一日游

## 8. 横店影视城

## 9. 西溪湿地

## 13. 江郎山·廿八都

## 15. 溪口-滕头

宁波市奉化区溪口镇，距宁波市区约35公里。

### 🏷特色
溪口-滕头景区由溪口古镇和滕头村两部分组成。溪口是蒋介石、蒋经国父子的故乡，拥有丰富的民国历史文化遗存。蒋氏故居（丰镐房）是蒋介石出生和成长的地方，建筑中西合璧。雪窦寺是弥勒菩萨道场，寺内的弥勒大佛高56.74米，是全球最高的坐姿铜制弥勒佛像。千丈岩瀑布落差186米，气势磅礴。雪窦山被誉为"四明第一山"，自然风光秀丽。滕头村是全球生态500佳、世界十佳和谐乡村，以生态农业和乡村旅游闻名，展示了中国新农村建设的成就。景区将历史文化、自然风光和乡村体验完美融合。

## 16. 鲁迅故里·沈园

### 📍位置
绍兴市越城区鲁迅中路，位于绍兴市中心。

### 🏷特色
鲁迅故里是鲁迅先生青少年时期生活过的地方，包括鲁迅故居、鲁迅祖居、三味书屋、百草园等景点，完整保留了清末民初绍兴的建筑风貌和生活场景。百草园是鲁迅童年时期的乐园，《从百草园到三味书屋》中描述的碧绿的菜畦、光滑的石井栏至今犹在。三味书屋是鲁迅少年时期读书的私塾，课桌上刻着的"早"字激励了无数学子。沈园是南宋时期的一座园林，因陆游与唐婉的爱情悲剧而闻名千古，园内墙壁上刻着两首著名的《钗头凤》。绍兴是鲁迅的文学故乡，从咸亨酒店到孔乙己，从社戏到闰土，鲁迅笔下的绍兴风情在这里一一呈现。夜晚的沈园有精彩的堂会表演，可以欣赏越剧、绍剧等地方戏曲。

## 17. 刘伯温故里

### 📍位置
温州市文成县，距温州市区约100公里。

### 🏷特色
刘伯温故里景区是为纪念明朝开国元勋刘基（刘伯温）而建立的文化旅游景区，由刘基庙、百丈漈、铜铃山等景点组成。刘基庙是全国规模最大的刘伯温纪念建筑，始建于明代，庙内塑有刘伯温坐像，展示了这位"三分天下诸葛亮，一统江山刘伯温"的传奇一生。百丈漈是全国单级落差最高的瀑布，总落差约353米，分为三漈，其中一漈落差207米，飞流直下，气势如虹。铜铃山国家森林公园以壶穴奇观著称，十二门瀑布连环跌落，形成独特的地质景观。文成县是浙江省森林覆盖率最高的县之一，生态环境极佳，空气清新，是天然的森林氧吧。

## 18. 缙云仙都

### 📍位置
丽水市缙云县仙都街道，距缙云县城约10公里。

### 🏷特色
缙云仙都是以峰岩奇绝、山水神秀为特色的国家级风景名胜区，被誉为"桂林之秀、黄山之奇、华山之险"。鼎湖峰是仙都的标志性景观，高170.8米，是世界上最高的石笋，状如春笋拔地而起，直刺苍穹。峰顶有天然小湖，传说是轩辕黄帝在此炼丹升天时留下的。黄帝祠宇是祭祀轩辕黄帝的圣地，每年清明节举办的祭祀大典规模宏大。朱潭山是仙都最佳的摄影点——清晨薄雾中，一位老农牵牛过桥，鼎湖峰倒映在平静的溪面上，这幅画面被无数摄影师拍摄，成为中国最美田园风光之一。小赤壁的悬崖峭壁上留有古代栈道遗迹，行走其间惊险刺激。仙都的美在于自然与人文的和谐统一。

## 19. 台州府城

### 📍位置
台州市临海市，位于台州市北部。

### 🏷特色
台州府城是中国保存最为完好的古城墙之一，始建于东晋，全长约6000米，素有"江南长城"之称。城墙依山就势，逶迤曲折，其形制与北京八达岭长城相似。巾山西麓的龙兴寺始建于唐代，是日本佛教天台宗的重要发祥地之一。紫阳古街是府城内保存最完好的历史街区，全长约1080米，两侧分布着大量明清时期的店铺和民居，至今仍有大量原住民生活，是真正的"活着的古城"。临海还是中国"美食之城"，蛋清羊尾、海苔饼、麦虾等特色小吃让人垂涎。台州府城不同于其他古城的商业化，保留了浓郁的生活气息和市井烟火。

## 20. 南湖

### 📍位置
嘉兴市南湖区南溪路1号，位于嘉兴市中心南部。

### 🏷特色
嘉兴南湖是中国共产党诞生地之一，1921年中共一大在南湖的一艘游船上完成了最后的议程，庄严宣告了中国共产党的诞生，南湖红船成为中国革命源头的象征。南湖景区以南湖红船和南湖革命纪念馆为核心，南湖革命纪念馆由"一船红天下"的红船造型建筑组成，馆内运用多媒体技术全面展示了中国共产党从诞生到壮大的光辉历程。南湖本身也是一处优美的城市湖泊，面积约624亩，湖中有湖心岛、小瀛洲等岛屿，烟雨楼矗立于湖心岛上，始建于五代，因唐代诗人杜牧"南朝四百八十寺，多少楼台烟雨中"而得名。湖畔的揽秀园、勺园等园林景色优美。

## 21. 双龙风景旅游区

### 📍位置
金华市婺城区罗店镇，距金华市区约15公里。

### 🏷特色
双龙风景区以溶洞景观闻名，核心景点双龙洞是中国最具代表性的溶洞之一。双龙洞因洞口两侧的钟乳石形似龙头而得名，洞内分为内外两洞，外洞宽敞明亮，内洞幽深神秘。进入内洞需要仰卧在小船上，通过一条仅容小船通过的水道，脸几乎贴着洞顶岩石，体验独特而刺激。叶圣陶先生的《记金华的双龙洞》是小学语文课文中的经典篇目，使双龙洞闻名全国。冰壶洞与双龙洞相连，洞内有一条落差约20米的瀑布，是国内溶洞中罕见的洞内瀑布。朝真洞是双龙洞群中海拔最高的溶洞，洞内石笋林立。景区内的黄大仙宫是道教名胜，黄大仙信仰在港澳和东南亚华人中影响深远。

## 22. 云和梯田

### 📍位置
丽水市云和县崇头镇，距云和县城约10公里。

### 🏷特色
云和梯田是华东地区最大的梯田群，始建于唐代，已有1000多年历史，海拔跨度200-1400多米，垂直高度超过1200米，拥有梯田700多层。云和梯田最美的时刻是清晨——当第一缕阳光穿透云海洒在灌满水的梯田上，层层叠叠的水面如镜面般反射着金色光芒，云雾缭绕间梯田若隐若现，宛如仙境。5-6月的灌水期和10-11月的稻谷成熟期是最佳观赏季节。景区内的白银谷是一条原始山谷，溪水潺潺，古树参天，沿途散布着几座百年古村落。坑根石寨是一座用石头砌成的古老村落，保存着原始的农耕生活方式。云和还是"中国木制玩具城"，景区内有木玩主题体验区。

### 🗺推荐旅游路线

**一日游路线：**
九曲云环观景台（日出/云海）→ 白银谷 → 坑根石寨（午餐）→ 梯田徒步 → 七星墩观景台（日落）

**深度游路线：**
Day 1：云和梯田全天游（日出+云海+梯田徒步+日落）
Day 2：云和湖 → 仙宫湖 → 赤石民宿小镇
Day 3：缙云仙都一日游

| 丽水缙云烧饼 | 丽水名小吃，梅干菜肉馅 | 梯田景区小摊 |
