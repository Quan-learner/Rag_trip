## 1. 秦始皇帝陵博物院（兵马俑）

| 项目 | 信息 |
|------|------|
| **所在城市** | 西安市临潼区 |
| **评定年份** | 2007年 |
| **景区类型** | 世界文化遗产 |
| **建议游玩时间** | 3-4小时 |
| **门票** | 120元 |

### 🗺️ 推荐路线

- 检票口 → 一号坑（最大最壮观！8000个兵马俑）→ 三号坑（指挥部）→ 二号坑（精品展示，跪射俑、铜车马）→ 铜车马展厅 → 秦始皇帝陵

### 🍜 附近美食

| 美食 | 推荐地点 | 说明 |
|------|----------|------|
| biangbiang面 | 临潼/西安市区 | 陕西名面，宽如裤带 |
| 肉夹馍 | 西安市区 | "中式汉堡" |
| 羊肉泡馍 | 西安老孙家/同盛祥 | 西安灵魂美食 |

### 💡 实用建议

- 一定要请导游讲解，否则只看到"一堆泥人"
- 铜车马是镇馆之宝，别错过
- 从西安坐游5路（306路）公交直达

## 2. 华清宫景区

| 项目 | 信息 |
|------|------|
| **所在城市** | 西安市临潼区 |
| **评定年份** | 2007年 |
| **景区类型** | 历史文化（皇家温泉） |
| **建议游玩时间** | 2-3小时 |
| **门票** | 120元 |

### 🗺️ 推荐路线

- 津阳门 → 唐御汤遗址博物馆（海棠汤、莲花汤）→ 西安事变旧址（五间厅）→ 骊山 → 晚上看《长恨歌》演出（强烈推荐！）

### 🍜 附近美食

| 美食 | 推荐地点 | 说明 |
|------|----------|------|
| 临潼石榴/火晶柿子 | 临潼特产 | 季节性水果 |
| 石子馍 | 临潼小吃 | 古老的烹饪方式 |

### 💡 实用建议

- 建议与兵马俑安排在同一天
- 《长恨歌》实景演出非常震撼，提前购票
- 骊山上有烽火台（烽火戏诸侯典故）

## 5. 华山风景区

| 项目 | 信息 |
|------|------|
| **所在城市** | 渭南市华阴市 |
| **评定年份** | 2011年 |
| **景区类型** | 五岳之西岳 |
| **建议游玩时间** | 1-2天 |
| **门票** | 旺季：160元；淡季：100元 |

### 🗺️ 推荐路线

**经典一日游（索道上下）：**
- 西峰索道上 → 南峰（最高峰2154.9m）→ 东峰 → 中峰 → 北峰 → 北峰索道下

**两日徒步游：**
- Day 1：玉泉院 → 北峰（千尺幢、百尺峡、苍龙岭）
- Day 2：北峰 → 东峰（看日出）→ 南峰 → 长空栈道 → 西峰索道下

### 🍜 附近美食

| 美食 | 推荐地点 | 说明 |
|------|----------|------|
| 华阴老腔 | 华阴市 | 非遗表演 |
| 大刀面 | 华阴市 | 宽面大碗 |
| 踅面 | 华阴市 | 中国最古老的方便面 |

### 💡 实用建议

- 长空栈道是华山最刺激的体验，恐高者慎选
- 西峰索道是最佳上山方式
- 夜爬华山看日出是经典体验

## 🗺️ 陕西5A景区联游推荐路线

### 经典西安三日游
```
Day 1: 兵马俑 → 华清宫 → 《长恨歌》
Day 2: 城墙骑行 → 碑林 → 钟鼓楼 → 回民街
Day 3: 大雁塔 → 大唐芙蓉园 → 大明宫
```

### 陕北红色+自然五日游
```
Day 1-2: 西安
Day 3: 延安（宝塔山、杨家岭、枣园）
Day 4: 壶口瀑布
Day 5: 乾坤湾 → 返程
```

### 华山+西安四日游
```
Day 1-2: 华山（日出）
Day 3-4: 西安市区
```
